# 八字干支合冲刑害总论

八字学习从入门到进阶，最容易卡住的一关，就是干支之间到底“怎么作用”。很多初学者把合冲刑害当成静态表格去背，真正进盘时却不知道哪个先看、哪个算真、哪个只是象。

《渊海子平》立局，《三命通会》广采诸法，《子平真诠》强调取用，《滴天髓》重神气流行。把这些思路综合起来，干支作用关系的核心不在于背口诀，而在于判断它是否真实改变了命局气机。

## 一、为什么要单独学习干支关系

干支关系影响三件关键事情：

1. 日主旺衰是否被改写
2. 格局主轴是否被扶成或破坏
3. 大运流年进入命局时，会触发什么变化

因此，合冲刑害不是附加信息，而是命局结构的一部分。

## 二、专业判断的基本顺序

建议按照以下顺序看：

1. 先看月令与全局气势
2. 再看天干透出之间是否有合化、争合、妒合
3. 再看地支是否成六合、三合、三会、六冲、刑害破
4. 最后再看这些关系有没有真正改变用神与格局

很多人一上来看到“甲己合”就断合化，或者看到“子午冲”就断婚变，这都是脱离全局的错误用法。

## 三、天干五合的真正判断

### 1. 五合关系

- 甲己合土
- 乙庚合金
- 丙辛合水
- 丁壬合木
- 戊癸合火

### 2. 合有三层含义

1. 合而有情：表示关系、牵引、合作、依附
2. 合而羁绊：表示被拖住、被牵制、节奏受限
3. 合而化气：表示真正改变化机与五行归属

### 3. 什么时候才谈“化”

天干合化不是看见五合就算，需要至少满足以下条件中的关键部分：

1. 得时令支持
2. 地支有根有气
3. 原局不被强力破坏
4. 命局整体气势有趋向

若只有天干表面相合，但无令无根，多数只论“合”，不轻论“化”。

### 4. 争合与妒合

当天干出现一干多合，或两干争一干时，说明关系复杂，牵引不专。命理上常见于人际、婚姻、资源分配、职业角色冲突等主题。

## 四、地支六合与三合三会

### 1. 六合

- 子丑合土
- 寅亥合木
- 卯戌合火
- 辰酉合金
- 巳申合水
- 午未合土

六合在命理上常表示关系建立、资源连接、条件配对，但不一定表示化气成功。很多时候，六合更多体现“绊住”“牵扯”“相互需要”。

### 2. 三合局

- 申子辰合水
- 亥卯未合木
- 寅午戌合火
- 巳酉丑合金

三合是强关系，若得令、有透干呼应，往往能明显改变全局气势。它对旺衰与格局影响很大，尤其在从格、化气格、食伤生财格、官印相生格中非常关键。

### 3. 三会局

- 亥子丑会水
- 寅卯辰会木
- 巳午未会火
- 申酉戌会金

三会比三合更接近季节之气，力量常更偏环境化、整体化。若三会成势，命局常带明显的五行偏向。

## 五、六冲不是只有“破坏”

### 1. 六冲关系

- 子午冲
- 丑未冲
- 寅申冲
- 卯酉冲
- 辰戌冲
- 巳亥冲

### 2. 六冲的本质

六冲表示动、离、变、开、散、对抗。它的意义取决于命局所处阶段和所问主题。

### 3. 常见表现

- 在原局中：表示结构不稳、人生节奏多变、关系张力大
- 在大运流年中：表示触发变化、搬迁、岗位调整、关系转折
- 对用神：用神被冲，要看是冲坏还是冲动

有些命局久静不发，恰恰需要冲来启动；有些原本稳定的结构，则被冲后失衡。所以不能机械地把冲等同于坏。

## 六、刑、害、破的细腻层次

### 1. 相刑

相刑多主内耗、别扭、曲折、反复、隐性代价。常见有：

- 寅巳申三刑
- 丑未戌三刑
- 子卯相刑
- 辰午酉亥自刑

刑的特点不是爆发式，而是持续性的摩擦和扭曲，常在心理、关系、制度压力中体现得更明显。

### 2. 相害

六害多主暗损、误解、关系不顺、利益不一致。它不像冲那么外显，却常常在婚姻、合作、职场中表现为“表面没翻脸，实际已难同心”。

### 3. 相破

破多主结构松动、信用折损、边界被破。它的表现比冲更轻，但在细节层面很重要，尤其影响稳定性与持续性。

## 七、合冲刑害如何落到实际命理判断

### 1. 对旺衰的影响

如果地支形成明显会局、合局，日主与十神的旺衰必须重算。旺衰不是看单字数量，而是看最后形成了什么气。

### 2. 对格局的影响

例如官印相生格，若印星被冲破，格局承接力就下降；食伤生财格若财星被合绊，也会出现“有才路但变现慢”的情况。

### 3. 对婚姻与关系的影响

夫妻宫、配偶星、日支若逢冲刑害，确实常见关系波动，但必须结合原局结构与用神判断，不能见冲就断离，见合就断好。

### 4. 对大运流年的影响

大运流年进入命局时，最重要的不是背一条“某冲某”，而是判断：

1. 冲的是用神还是忌神
2. 合的是资源还是负担
3. 刑害是否引动原局旧病
4. 这种作用是暂时触发，还是改变阶段主轴

## 八、几个高频误区

### 1. 见合必喜

合可能是成，也可能是绊。尤其财星、官星被合时，常常意味着“有联系，但未必能自由发挥”。

### 2. 见冲必凶

冲也可能是破旧立新，特别在迁移、职业转换、创业、换环境上，冲往往是必要动作。

### 3. 把刑害当作次要信息

很多盘真正难受的地方，不是大冲大克，而是长期刑害造成的内耗。

### 4. 合化滥用

合与化是两回事。只有命局整体支持时，才谈化气成立。

## 九、学习建议

训练时不要只背表格，建议每看一个八字都回答四个问题：

1. 这个局里最强的关系是什么
2. 这种关系改变了哪股五行气势
3. 它对用神是帮助还是破坏
4. 进入大运流年后，会被谁再次触发

只要这四个问题答得清，干支关系就不会停留在口诀层面。
