# 八字命理入门

## 什么是八字？

八字，又称四柱八字，是中国传统命理学的重要组成部分。八字由一个人出生的年、月、日、时四个时间要素组成，每个要素由天干和地支两个字表示，共八个字，因此称为"八字"。

## 基本概念

### 天干

天干共有十个：甲、乙、丙、丁、戊、己、庚、辛、壬、癸。

### 地支

地支共有十二个：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥。

## 四柱

- **年柱**：代表祖上、父母，反映早年运势
- **月柱**：代表兄弟、朋友，反映青年时期
- **日柱**：代表自己和配偶，是命局的核心
- **时柱**：代表子女，反映晚年运势

## 五行

五行包括：木、火、土、金、水。五行之间存在相生相克的关系：

- **相生**：木生火、火生土、土生金、金生水、水生木
- **相克**：木克土、土克水、水克火、火克金、金克木
