# 六爻事业与求职占断

事业卦的复杂性，在于它通常不是单一问题。求职看职位，升职看层级，跳槽看变化，创业看风险与资源，回到六爻结构里，往往同时牵涉官鬼、父母、世爻、应爻，甚至财爻与子孙爻。

因此，职业类卦如果只抓一个用神，往往会断得过窄。专业判断更像搭一个结构：职位在不在、制度通不通、自己承不承得住、对方愿不愿意接。

这类题目在实战里最常见的错误，是把“有没有结果”“值不值得接”“入职后稳不稳”“该不该跳”压成同一卦。职业题一旦问题过大，官鬼、父母和世应都容易被读混。起卦前先把当前决策点收敛清楚，会比重复追问更稳。可先读：[六爻重复问卦与提问规范](repeated-divination-guidelines.md)

## 一、事业卦的主轴是什么

职业类问题最常见的主轴是：

1. 官鬼
2. 父母
3. 世爻
4. 应爻

### 四者分工

- 官鬼：岗位、职位、责任、编制、管理层级、压力
- 父母：合同、手续、简历、证照、制度、流程、文书面
- 世爻：自己当前的能力、意愿、承受度
- 应爻：公司、单位、岗位方、外部环境反应

## 二、不同题目的取用重点

### 1. 求职

以官鬼为主，看职位是否存在与是否对自己开放；兼看父母爻判断简历、面试、录用流程。

### 2. 升职

重点仍在官鬼，但更强调官鬼能否生世、扶世，或世能否承官。若官旺而克世，多数代表责任增大，不一定是舒适的晋升。

### 3. 跳槽

跳槽卦一定要看“旧局能不能破、新局能不能接”。冲在此类卦里，经常不是坏事，而是启动变化的必要动作。

### 4. 创业

创业不只看官鬼，还要看财爻与子孙。官鬼代表经营压力与规则，财爻代表资源与回报，子孙代表产品与打法。

## 三、官鬼怎么读才专业

### 1. 官鬼旺相

表示职位、岗位、组织结构本身有力，通常说明机会是真实的，不是空头承诺。

### 2. 官鬼空破

常见于：

- 岗位并未最终敲定
- 承诺不落地
- 有消息但不稳定
- 单位内部尚未明确

### 3. 官鬼生世与克世

- 官鬼生世：岗位对自己有接纳和提升作用
- 官鬼克世：岗位压力大，管理强，规则重，未必轻松

这并不是简单吉凶，而是职业性质判断。很多高位岗位，本就带克世之象。

## 四、父母爻在职业卦中的重要性

父母爻在事业卦里非常关键，尤其是现代职场环境下。

它常常代表：

1. 简历与资历
2. 合同与文件
3. 面试流程
4. 证照、考试、审批
5. 公司制度与组织程序

### 常见判断

- 父母旺动：制度流程推进明显
- 父母生官：资历、手续有助于职位落地
- 父母空破：流程拖延、通知反复、文书迟滞

## 五、世爻与应爻的职业关系

职业卦里的世应，通常比感情卦更偏现实互动，而非情绪关系。

### 典型结构

1. 世应相生：双方匹配度较高
2. 应生世：单位方更认可自己
3. 世生应：自己投入更多，争取更强
4. 世应相冲：岗位方向、价值判断、节奏不一致
5. 应动世静：外部变动先到来
6. 世动应静：自己更主动寻求变化

## 六、跳槽卦的特别判断法

跳槽卦常常不能用“稳就是好、动就是坏”的思路去看。

### 关键问题有两个

1. 原岗位是否还值得留
2. 新岗位是否能接得住自己

### 常见信号

- 原官鬼受制、新官鬼有力：适合转
- 世爻已衰、原局持续克世：不变反更耗
- 应爻有情、新官鬼得生：新平台有承接力

## 七、创业卦怎么看

创业是职业类卦里最复杂的一种，因为它把职位、收益、风险、资源都叠在一起了。

### 创业常看四条线

1. 财爻：资金与回报
2. 子孙：产品、方案、打法
3. 官鬼：规则、经营压力、责任
4. 世爻：自己是否承得住节奏与风险

### 结构判断

- 子孙生财：打法能变现
- 财旺世弱：项目看起来大，但自己未必能稳住
- 官鬼旺：规则、风险、经营责任大
- 父母旺：手续、资质、框架面较稳

## 八、应期判断

职业卦的应期，不一定是“拿到结果”的时间，也可能是“开始谈”“开始动”“开始办手续”的时间。

### 常见抓手

1. 官鬼逢值
2. 父母逢值
3. 官鬼或父母出空
4. 世应逢合或逢冲而启动
5. 动爻化进，表示进程推进

## 九、常见误区

### 1. 只看官鬼，不看父母

职位存在，不代表手续能顺利落地。

### 2. 官鬼克世就断不好

高压岗位、体制岗位、管理岗位，本来就带较强约束，并不等于一定坏。

### 3. 跳槽见冲就怕

跳槽题中，冲常常是破旧局的必要动作。

### 4. 创业卦按求职卦看

创业不是单纯“有没有岗位”，而是“能不能建立新结构”，所以必须把财、子孙、官鬼一起纳入。

## 十、实务分析模板

职业卦建议先总结四句：

1. 岗位或机会是否真实存在
2. 自己是否接得住这个位置
3. 阻力来自流程、环境还是能力不匹配
4. 变化会先体现在消息、手续还是正式结果
