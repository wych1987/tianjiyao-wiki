# 六爻占卜实战案例详解

## 第一章 婚姻感情类案例

### 案例1：测婚姻是否和顺

**占问事项**：王女士，35岁，结婚8年，近期与丈夫矛盾频发，测婚姻能否和顺

**起卦时间**：2023年5月15日10时28分（农历三月廿六日巳时）

**起卦过程**：采用铜钱摇卦法，三枚乾隆通宝摇六次：
第一次：两字一背（少阳）
第二次：三字（老阴）
第三次：两背一字（少阴）
第四次：三背（老阳）
第五次：两字一背（少阳）
第六次：三背（老阳）

**卦象呈现**：
主卦：火泽睽
变卦：火水未济

```
主卦：火泽睽
▅▅▅▅▅ 父母巳火
▅▅ ▅▅ 兄弟未土
▅▅▅▅▅ 子孙酉金 世
▅▅ ▅▅ 兄弟丑土
▅▅▅▅▅ 官鬼卯木
▅▅▅▅▅ 父母巳火 应

变卦：火水未济
▅▅▅▅▅ 父母巳火 应
▅▅ ▅▅ 兄弟未土
▅▅▅▅▅ 子孙酉金
▅▅ ▅▅ 父母午火 世
▅▅▅▅▅ 兄弟辰土
▅▅ ▅▅ 官鬼寅木
```

**装卦分析**：
- 地支：巳、未、酉、丑、卯、巳
- 六亲：父母、兄弟、子孙、兄弟、官鬼、父母
- 六神：青龙、朱雀、勾陈、螣蛇、白虎、玄武
- 世应：世在四爻酉金，应在初爻巳火

**详细分析**：
1. 用神选取：女测婚姻以官鬼为用神，本卦官鬼卯木临二爻
2. 旺衰判断：卯木在巳月得生，但卦中金重克木，用神偏弱
3. 动爻分析：三爻兄弟丑土动化父母午火，兄弟动主争执
4. 六神分析：官鬼临白虎，主丈夫脾气急躁；应爻父母临玄武，主对方心思不明
5. 世应关系：世克应，主求测人想要控制对方，但酉金克卯木，形成对抗

**断卦结论**：婚姻暂时不和，但不会离婚，需要双方各退一步

**应验反馈**：2023年8月反馈，经过婚姻咨询，双方关系有所改善，但仍有小摩擦

**经验总结**：测婚姻要重点看世应关系和官鬼、妻财的状态，兄弟动多为争执之象

### 案例2：测何时结婚

**占问事项**：李先生，28岁，恋爱三年，测何时能结婚

**起卦时间**：2023年3月8日14时30分

**起卦过程**：数字起卦法，取数3、8、14，得：
上卦：3为离
下卦：8为坤
动爻：14÷6余2，二爻动

**卦象呈现**：
主卦：火地晋
变卦：火风鼎

变卦：火风鼎

▅▅ ▅▅ 父母未土 应
▅▅▅▅▅ 兄弟酉金
▅▅▅▅▅ 兄弟酉金
▅▅▅▅▅ 官鬼巳火 世
▅▅ ▅▅ 父母未土

**详细分析**：（限于篇幅，简略分析）

**断卦结论**：2024年春季有望结婚

**应验反馈**：2024年4月举行婚礼

**经验总结**：测婚期要看用神出现、旺相之时，兼看世应关系

## 第六章 案例分析总结

### 共性规律提炼

1. **用神选取规律**
   - 婚姻：男测妻看财爻，女测夫看官鬼
   - 事业：官鬼为职位，父母为单位
   - 财运：妻财为钱财，子孙为财源
   - 健康：官鬼为病，子孙为医

2. **旺衰判断要点**
   - 月建日辰最重要
   - 动爻变爻影响大
   - 五行生克是基础

### 判断要点归纳

1. **世应关系**：世为自己，应为对方，生克定关系
2. **动变分析**：动爻主事之始，变爻主事之终
3. **六神辅助**：结合六神看事体性质
4. **卦象综合**：主卦现状，变卦结果

### 常见错误分析

1. **用神选取错误**：事体分类不清
2. **旺衰判断片面**：忽视月日影响
3. **过分看重一爻**：缺乏整体观
4. **忽略应期判断**：只说吉凶不论时间

### 准确率提升方法

1. **基础扎实**：熟练掌握五行生克、八卦类象
2. **综合分析**：多角度验证判断

4. **心态端正**：客观分析，避免主观倾向
