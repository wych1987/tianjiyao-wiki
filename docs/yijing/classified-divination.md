# 六爻分类占断总论

六爻之学，难在“同一套装卦结构，要对应不同问题”。起卦装卦只是共同底盘，真正体现断卦水平的，是面对不同占题时能否迅速抓住正确用神、主次关系与判断顺序。

《增删卜易》重在用神，《卜筮正宗》长于分类占断，《易隐》则重视神机与变通。若把这三类思路综合起来，分类占断可以归结为一句话：先辨题目，再立主神，后看成败，最后推应期。

但在进入分类之前，还有一步经常被忽略：先把问题收敛成“当前阶段的一件事”。如果一上来就把结果、过程、时间、对方态度和下一阶段决策混在同一卦里，就算大类分对了，后面的主线也仍然会乱。可先读：[六爻重复问卦与提问规范](repeated-divination-guidelines.md)

## 一、分类占断为什么重要

同样一卦，占婚姻、占合同、占病情，重点完全不同。若只会机械看“哪一爻旺，哪一爻衰”，而不知道不同题目该抓什么，就很容易出现判断失焦。

分类占断的价值在于：

1. 帮助快速确定主用神
2. 区分主线与辅助信息
3. 让世应关系真正落到具体场景
4. 让应期推断不再脱离事件本身

## 二、通用判断顺序

不论占什么题目，建议都先遵守同一套顺序：

1. 明确问的到底是结果、过程还是时间
2. 选择主用神
3. 确认世应关系
4. 看用神旺衰与是否受制
5. 看动爻、合冲、旬空、月破
6. 判断是否能成、何时应、以何种方式应

很多误判，不是不会看，而是把顺序弄反了。

## 三、婚姻感情占断

### 1. 取用原则

- 男问感情或婚姻，以妻财为主
- 女问感情或婚姻，以官鬼为主
- 世爻看自己状态，应爻看对方态度与关系接口

### 2. 重点判断项

1. 用神是否旺相得气
2. 世应是否相生、相合、相冲、相克
3. 用神是否逢空、破、墓、绝
4. 兄弟、子孙是否构成干扰

### 3. 常见象义

- 世应相生：双方有互动，有继续空间
- 世应相冲：关系紧张、节奏失衡、聚少离多
- 用神得月日扶：关系基础仍在
- 用神空破：关系有名无实，或心态飘忽未定
- 兄弟旺动：竞争者、朋友介入、现实杂音变大

### 4. 实战判断原则

感情卦最常见的误区，是只盯着官财，而忽略世应。事实上，很多关系不是没有对象，而是没有接口。世应无情、相背、互不承接，即便用神不差，关系也未必能落实。

## 四、求财与交易占断

### 1. 取用原则

- 一般求财，以妻财为主
- 问财源、盈利方式，可兼看子孙
- 问合作交易，需同时看应爻与父母爻

### 2. 重点判断项

1. 财爻旺不旺
2. 世爻能不能承财
3. 兄弟是否夺财
4. 官鬼是否形成制度、债务、税费、风控压力

### 3. 常见象义

- 财旺世旺：有能力得财并承财
- 财旺世弱：有机会但未必拿得住
- 财衰而子孙旺：想法多、动作多，但收益落实慢
- 兄弟旺动：利润被分流、竞争激烈、支出偏多
- 财伏：钱未显、账未到、项目未结算

### 4. 关于投资卦

投资不应只看“财有没有”，更要看“子孙能不能生财，世爻能不能承压”。子孙在财卦中常有两个功能：一是财路、策略、产品；二是风险释放。子孙过弱，往往意味着节奏把握差；子孙过旺又无制，也可能代表过度乐观。

## 五、事业与职位占断

### 1. 取用原则

- 求职、升职、岗位变化，以官鬼为主
- 合同、文书、流程、录用通知，多兼看父母
- 单位态度、对方意向，多结合应爻

### 2. 重点判断项

1. 官鬼是否得令得气
2. 世爻是否能承职位与压力
3. 父母爻是否顺畅，表示手续与制度面通不通
4. 应爻是否生合世爻，表示单位是否有接纳意愿

### 3. 常见象义

- 官鬼旺而生世：职位机会来临
- 官鬼旺而克世：岗位有压力，可能难以承受
- 父母动生官：文件、制度、资历有帮助
- 官鬼空破：职位承诺不足，消息未定，岗位虚而不实

### 4. 跳槽与变动的特别判断

问跳槽，冲未必是坏事。若原岗位已形成压制，冲往往代表离开旧局；若新岗位用神有根而应爻有情，则冲反而是启动变动的必要动作。

## 六、疾病与健康占断

### 1. 取用原则

- 官鬼多看病邪、病灶、压力源
- 世爻看身体承受度与命主本身
- 子孙看医药、治疗、恢复力与解除病因的能力

### 2. 重点判断项

1. 官鬼旺衰与是否克世
2. 世爻是否有根、有扶、有救
3. 子孙能否制鬼、护世
4. 白虎、玄武、腾蛇等六神是否补出病象细节

### 3. 常见象义

- 官鬼旺克世：病情压力重
- 子孙发动制鬼：有治疗机会，恢复希望增加
- 官鬼空而反复：病名未明、检查反复、症状漂移
- 白虎临鬼：外伤、炎症、手术、血光类风险要提高警惕

### 4. 使用边界

六爻疾病占断适合作为趋势判断和风险提醒，不应替代正规医疗诊断。专业写作中必须明确这一点，既体现审慎，也符合知识库规范。

## 七、考试、证件与文书占断

### 1. 取用原则

- 父母主文书、试卷、资格、录取信息
- 官鬼主名次、编制、结果中的“位阶”因素
- 世爻主本人状态与发挥

### 2. 重点判断项

1. 父母是否旺相
2. 世爻是否得扶，能否接住父母爻之生
3. 官鬼是否成局，表示是否真正拿到名额或位置
4. 空亡、月破是否表示信息延迟或流程卡顿

### 3. 延伸专题

若继续深入，建议单独阅读 [六爻考试升学占断](exam-divination.md)，并配合 [六爻应期判断](response-timing.md) 一起练习消息应、结果应与录取应的区别。

## 八、出行、迁移与行程占断

### 1. 取用原则

- 世爻看自己
- 应爻看目的地或对接方
- 父母可看票据、证照、路程文书
- 官鬼可看风险、阻碍、检查、规则压力

### 2. 核心判断

- 用冲而动，多主成行
- 世应有情，多主对接顺利
- 父母空破，多主证件、手续、行程信息不稳
- 官鬼旺动，多主延误、管制、风险、奔波疲惫

## 九、失物与寻人占断

### 1. 取用原则

- 一般失财物，以财爻为主
- 证件文书类，以父母为主
- 宠物、小孩、学生等，也常参考子孙

### 2. 判断思路

失物卦关键是“还在不在”“近不近”“动没动”“能不能回”。此时世应之外，还要重视卦宫方位、内外卦、六神、冲合与伏神。

## 十、分类占断中的几个共性误区

### 1. 用神取对了，但主线没立住

例如问婚姻却只盯官财，不看世应与兄弟扰动；问求财却只盯财爻，不看世能不能承财。

### 2. 不区分“有没有”和“能不能成”

用神出现，只代表题目有对应事项；能否成，要看旺衰、生克、合冲、空破。

### 3. 把分类占断当口诀堆砌

真正的分类占断不是背条目，而是知道为什么某个题目该抓某条主线。
