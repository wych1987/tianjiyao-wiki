# 六爻婚姻感情占断

婚姻感情类问题，是六爻实战中最常见、也最容易误判的一类。原因不在于规则不够，而在于关系问题天然复杂，既有主观情绪，也有现实条件，还常常夹杂第三方因素、时间差和反复试探。

《增删卜易》重用神，《卜筮正宗》善分类，《易隐》强调神机流动。落实到婚姻卦中，可以归纳成一句话：先看有没有对象与关系基础，再看双方能否对接，最后判断成败、变数和时间。

## 一、婚姻卦为什么不能只看官财

传统规则中，男问女以妻财为用，女问男以官鬼为用，这是正确的起点，但不是全部。现实中很多卦，用神并不差，可关系仍然难成，问题往往出在世应无情、兄弟扰动、用神空破或者关系节奏不一致。

因此，婚姻卦至少要同时看四条线：

1. 用神
2. 世应
3. 干扰爻
4. 应期

## 二、基本取用原则

### 1. 男问感情

- 以妻财为主用神
- 世爻代表自己
- 应爻代表对方态度或关系接口

### 2. 女问感情

- 以官鬼为主用神
- 世爻代表自己
- 应爻代表对方态度或关系接口

### 3. 特殊题目

- 问复合：主看原关系是否仍有牵引
- 问订婚结婚：主看关系能否落实到制度层面
- 问离婚：主看现有关系是否已失承接力
- 问第三者：官财之外，兄弟、应爻、伏神常有提示作用

## 三、世应是关系卦的第一坐标

世应不是可看可不看，而是婚姻卦最核心的互动轴。

### 常见关系模式

1. 世应相生：双方有回应，有继续空间
2. 世应相合：有牵引、有黏性，但要分清是成合还是绊合
3. 世应相冲：节奏不一致，关系有波动、争执或聚少离多
4. 世应相克：彼此消耗，难以真正承接
5. 世动应静：自己主动更多
6. 应动世静：对方态度变化更关键

### 实务判断

如果世应之间没有真实互动，就算官财旺，也容易变成“有其人，无其缘；有其心，无其局”。

## 四、官鬼与妻财在婚恋卦中的落点

### 1. 用神旺相

表示关系对象本身存在基础，有现实推动力，或者对方在这段关系中仍有明确位置。

### 2. 用神空破

常见于：

- 关系有名无实
- 对方态度未定
- 承诺漂浮不落地
- 人虽在，心不在

### 3. 用神入墓、伏藏

多主关系进入隐蔽、压抑、难公开、难推进的阶段，也常见于冷战、断联、旧情未了但暂难显化。

## 五、第三方因素怎么看

感情卦里的第三方，不一定就是“小三”，还可能是朋友、家人、工作、距离、现实安排。

### 常见信号

1. 兄弟旺动：竞争者、外部干扰、朋友掺入、资源分流
2. 父母旺动：家庭、长辈、文书、现实制度压力介入
3. 官鬼或财伏而另有同类爻旺动：关系内外不一，暗线较重
4. 玄武、腾蛇信息明显：隐情、反复、猜疑、模糊地带增加

## 六、不同题目的判断重点

### 1. 问能不能开始

重点看：

- 世应是否相生有情
- 用神是否出现且得气
- 是否有明显干扰爻压制

### 2. 问能不能复合

复合卦最重要的不是看“还喜欢吗”，而是看“旧关系是否还有重新对接的条件”。

重点看：

- 世应是否仍有牵引
- 用神是否被冲散、空破、化退
- 是否有动爻重新引出旧情结构

### 3. 问能不能结婚

结婚属于关系落地题，除了官财与世应，还要看父母爻，因为父母常主证件、仪式、家庭制度面。

### 4. 问会不会分开

不能只看冲克，还要看是否已经失去承接：

- 世应长期相克
- 用神衰而无救
- 干扰爻持续强势
- 动化退、化绝、化破

这些组合一起出现时，分离倾向才更明确。

## 七、应期判断

婚姻卦的应期，要分“关系变化时间”和“结果落实时间”。这两者常常不是同一天。

### 常见应期抓手

1. 用神逢值
2. 用神出空
3. 世应逢合或逢冲而动
4. 伏神被引出
5. 父母爻成局，表示手续与现实条件到位

### 实务提醒

关系卦尤其忌机械定时。感情问题受双方主观意愿影响很大，应期更多是“条件成熟点”，而不是绝对必发点。

## 八、常见误区

### 1. 男看财、女看官之后就结束

这是最典型的初学者误区。官财只是关系对象的入口，不是整段关系的全部。

### 2. 世应相合就断一定能成

合有时是牵引，有时是绊住。尤其用神空破时，常见“纠缠不断，但很难落实”。

### 3. 见兄弟就断第三者

兄弟也可能代表朋友、竞争、现实干扰、同温层影响，不宜一刀切。

### 4. 见冲就断分手

冲有时是打破僵局，也可能代表异地、节奏错位、阶段性摩擦，需结合整体结构判断。

## 九、案例式判断模板

分析婚姻卦时，可先按以下四句写：

1. 关系对象是否真实存在且有力
2. 双方是否有对接接口
3. 干扰来自哪里
4. 关系变化会先发生在何时、何层面

这四句若能写清，多数婚恋卦就不会断偏。
