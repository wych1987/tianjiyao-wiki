# 六爻求财与投资占断

财务类问题，是六爻中最容易“看见希望，也最容易看错重心”的一类。很多人只要看到财爻出现就判有财，或者财爻旺就判发财，忽略了世爻能不能承财、兄弟会不会分财、子孙能不能生财、应爻是不是代表真实交易条件。

真正专业的财卦，不是只看“有没有钱”，而是看“财从哪里来，能不能拿到，拿到后能不能守住”。

财务题还有一个高频误区，是把“这个项目能不能做”“多久能回款”“能不能赚钱”“赚到的钱能不能落袋”一次全问完。这样看起来具体，实际仍然是多层问题叠在一起。起卦前先把你当前最想判断的那一层收紧，六爻才更容易抓到主线。可先读：[六爻重复问卦与提问规范](repeated-divination-guidelines.md)

## 一、财卦的核心结构

财务问题通常至少要看四个点：

1. 妻财
2. 世爻
3. 子孙
4. 兄弟

若涉及交易、合作、签约、项目、客户，还要同时看应爻与父母爻。

## 二、取用原则

### 1. 求财

一般以妻财爻为主。它代表可得之财、项目收益、订单、货款、物资、客户资源等。

### 2. 投资与盈利模式

子孙爻常是关键辅助用神，因为子孙代表方法、产品、释放风险的能力，也代表“生财路径”。

### 3. 回款、合同、交易

除了财爻，还应看：

- 应爻：对方、客户、交易对手
- 父母爻：合同、票据、流程、制度条件

## 三、财爻旺不等于一定得财

### 1. 财旺世旺

这是较理想的结构，表示有财源且自己有承接能力。

### 2. 财旺世弱

表示机会存在，但自己可能拿不稳、接不住、管不好。现实中常见于：

- 项目过大超出能力范围
- 资金机会来了但执行力不足
- 钱看得见，却难真正落袋

### 3. 财衰世旺

表示自己有意愿、有动作，但当前市场、客户、项目本身不给力。

## 四、子孙在财卦中的地位

很多财卦若只看财爻，很容易断浅。子孙爻在财卦中往往代表：

1. 产品能力
2. 策略与方法
3. 风险释放能力
4. 财源发动点

### 常见结构

- 子孙生财：方法对路，财有来源
- 子孙旺而财弱：想法多、路径有，但收益转化慢
- 子孙衰弱：策略不足，执行断层，变现能力弱

## 五、兄弟为什么总是财卦重点

兄弟爻在财卦中常主竞争、分利、开支、合作方抽成、亲友借耗、成本上升等。

### 常见象义

1. 兄弟旺动：竞争强、利润被稀释、花费增加
2. 兄弟生应：对方条件更强，财利偏向外部
3. 兄弟合财：财被占住、被拖住、被分走

因此，财卦中看到兄弟旺，不一定是“破财”那么简单，很多时候是利润率下降而非完全无财。

## 六、不同财务题目的判断侧重

### 1. 问收入与赚钱机会

重点看财爻是否得气、世是否承财、子孙是否生财。

### 2. 问投资值不值得做

重点看：

- 财爻是否有落袋可能
- 子孙能否稳定生财
- 官鬼是否形成风险、监管、债务或不确定性

### 3. 问回款与收账

重点看：

- 财爻是否现形
- 应爻是否有情
- 父母爻是否表示流程与票据顺畅
- 玄武是否提示拖欠、隐瞒、赖账

### 4. 问项目合作

合作卦里财不只是收益，还可能是项目本体。此时必须同时看应爻和兄弟爻，判断合作是互补、竞争，还是利益分流。

## 七、官鬼在财卦中的双重角色

官鬼不只是不利因素。在财务问题里，官鬼可能代表：

1. 合规压力
2. 税务、制度、审查
3. 风险、负债、合同责任
4. 也可能是“职位带来的收入入口”

### 判断原则

- 官鬼克世：压力明显，做财要承担代价
- 官鬼护财：某些情况下制度反而保护财利
- 官鬼旺而财弱：重规则、轻利润，收益兑现慢

## 八、应期怎么抓

财卦应期关键看“财何时可动、何时可到、何时可落袋”。

### 常见抓手

1. 财爻逢值
2. 财爻出空
3. 财爻受冲而动
4. 子孙发动生财
5. 父母爻到位，表示流程完成

### 特别提醒

如果卦象显示“有财但慢”，应期往往不是看第一次发动，而是看财真正脱离空破、摆脱合绊的时间点。

## 九、投资卦最常见的误区

### 1. 只看财爻旺不旺

投资成败常常不在财爻本身，而在子孙和世爻。没有好的方法与承接能力，再旺的财也难久守。

### 2. 见兄弟就断破财

兄弟很多时候是成本、竞争、分润，不等于血本无归。

### 3. 见子孙旺就过度乐观

子孙旺可能代表创意足、执行多，也可能代表过度释放、过度乐观、过度扩张。

### 4. 忽略应爻

交易型财卦最忌忽略应爻。很多项目不是不能赚，而是对方不给配合、不履约、不给回款。

## 十、实务分析模板

财卦建议先用四句总结：

1. 财源是否真实存在
2. 自己能否承接这笔财
3. 最大阻力来自竞争、风险还是流程
4. 财是快得、慢得，还是有财无袋
