# 六爻应期案例专题

六爻应期最难的地方，不是记口诀，而是知道同一件事里，什么算“先应”，什么算“后应”，什么只是过程中的信号，什么才是真正定局。

这页不重复讲应期理论，而是用几种高频案例模型，把“消息应、动作应、结果应、出空应、冲开应、阶段成熟应”拆开来看。

如果你一看案例就想直接问“那我这件事到底哪天应”，更应该先把问题层次收紧。应期案例的价值，不是替你跳过前置判断，而是帮你知道同一件事里哪些该先问、哪些要等下一阶段再问。若还不确定同题和分层边界，可先读：[六爻重复问卦与提问规范](repeated-divination-guidelines.md)

## 一、先记住一个总原则

很多人问应期，一上来就在问日期；但更稳的顺序其实是：

1. 事情能不能成
2. 现在卡在哪里
3. 先应的是消息、动作还是结果
4. 最后再看何时应

如果顺序反了，应期通常会漂。

## 二、案例一：先有消息，后有结果

### 问题类型

求职者问：这家公司会不会录用我，什么时候有结果。

### 典型结构

1. 官鬼为职位信息，有动象
2. 父母爻为通知、文书，也有承接
3. 世爻暂弱，但不至于断局

### 判断重点

这种卦最容易出现“两段应”：

1. 先应在收到回复、面试通知、补材料
2. 后应在正式录用或入职落定

### 学习结论

如果不先区分“消息”和“结果”，就会以为自己应期判断错了。其实很多卦不是错，而是你把第一层应和最终定局混成了一层。

## 三、案例二：关系被拖住，等冲开才动

### 问题类型

问感情复合，对方会不会主动联系。

### 典型结构

1. 世应尚有关系
2. 关键爻被合住
3. 动而不实，当前有牵扯但推进慢

### 判断重点

这类卦常见：

1. 当前不是完全断开
2. 但事情被拖住，不会立刻落地
3. 需要等某个冲开点，关系才会进入新阶段

### 学习结论

感情题里的“合”不一定就是好，有时反而是拖住。真正的推进，反而发生在冲开之后。

## 四、案例三：出空以后才算真正应事

### 问题类型

问项目合作是否能签下来。

### 典型结构

1. 用神旬空
2. 表面上双方都有意愿
3. 但关键条件尚未落地

### 判断重点

这种卦常见一个误判：当事人觉得“已经谈得差不多了”，就以为快成；但卦里真正提示的是“事情未实”。

### 学习结论

用神空，不代表永远不成，但说明当下还未定。专业推期时，应重点观察出空、逢值、填实的时间点。

## 五、案例四：阶段不成熟，不能硬压近期日期

### 问题类型

问创业项目什么时候能真正起量。

### 典型结构

1. 用神当前弱
2. 月令背景不支持
3. 近处日时虽有小动，但不足以成局

### 判断重点

这种卦最大的问题，不是“有没有一点信号”，而是阶段成熟度根本不够。强行压到某一天，只会导致断期越来越乱。

### 学习结论

越大的事，越先看阶段；越小的事，才越适合收缩到具体日时。创业、调岗、搬迁、长期关系这种问题，很多都属于阶段成熟型。

## 六、案例五：先动后退，说明不是不应，而是反复应

### 问题类型

问资金回款何时到账。

### 典型结构

1. 财爻有动象
2. 但化退、受制或承接不稳
3. 中间可能先有承诺、后有拖延

### 判断重点

这类卦最容易出现“先说要给，再拖一轮”的情况。若只看第一次动作，就会误以为已经应完；若只看最终到账，又会漏掉前面的信号。

### 学习结论

应期有时不是单点，而是一串过程。真正成熟的判断，是能把不同层级的应区分开来。

## 七、案例六：疾病题更适合看转折，不适合机械追某一天

### 问题类型

问治疗何时见效，风险何时缓解。

### 典型结构

1. 官鬼、世爻、子孙之间形成拉扯
2. 当前仍在病势变化中
3. 结果不会在一个点上突然完全结束

### 判断重点

疾病题常见的应期，不是“某天彻底好”，而是：

1. 什么时候开始转轻
2. 什么时候指标有改善
3. 什么时候风险窗口过去

### 学习结论

疾病题更适合看转折和风险窗口，而不是机械压到某一天全好或全坏。

## 八、六种常见应期模型总结

读完上面的案例，可以把六爻应期大致分成六种常见模型：

1. 消息应：先有通知、回复、表态
2. 动作应：先有见面、提交、启动、接触
3. 结果应：最后才有正式定局
4. 出空应：事情原本未实，要等填实
5. 冲开应：事情被拖住，要等打破静局
6. 阶段成熟应：不是某天突然成，而是某阶段才具备承接条件
