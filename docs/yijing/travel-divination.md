# 六爻出行迁移占断

出行迁移类问题，在六爻中最容易被误判成“动就凶，静就吉”。其实正相反，这类问题的核心往往在于“动得成不成、动得顺不顺、动了之后有没有承接”。对旅行、出差、搬家、岗位调动、异地发展来说，冲与动很多时候恰恰是必要条件。

因此，出行卦判断时，不是先问吉凶，而是先问三个问题：

1. 能不能动
2. 动了有没有阻力
3. 动后落点稳不稳

## 一、主用神怎么抓

### 1. 世爻

世爻代表自己，是出行动作的主体。

### 2. 应爻

应爻代表目的地、对接方、外部环境、旅途中遇到的人事条件。

### 3. 父母爻

父母常主票据、证件、手续、车辆、路线文书、搬迁合同等，是现代出行卦里非常重要的一条线。

### 4. 官鬼爻

官鬼常主风险、阻碍、检查、交通规则、延误、事故、政策性限制。

## 二、什么叫“成行”

很多人问出行卦，真正想知道的是“这趟能不能成”。

### 常见有利结构

1. 世爻发动
2. 世应有情
3. 用冲而动
4. 父母爻顺畅
5. 官鬼不过分压制

### 常见不利结构

1. 世爻空破、无力
2. 父母空破，手续票据有问题
3. 官鬼旺动，阻力很重
4. 世应完全无情，目的地不承接

## 三、为什么这类卦要重视父母爻

在古法中，父母本就主舟车、文书、路引。放到今天，它对应的范围更广：

1. 证件
2. 机票车票
3. 行程信息
4. 租约与搬迁手续
5. 调令、通知、审批流程

很多出行问题，不是人不想动，而是父母爻不通，导致整个动作卡在手续层。

## 四、冲在出行卦里为什么常常是好事

六爻里“冲”本来就有动、开、离位、启动的含义。出行卦、搬迁卦、调动卦若一点不动，反而可能停在原地。

### 常见理解

- 世逢冲：自己要动身、要变化
- 应逢冲：目的地或外部条件在变化
- 父母逢冲：票据、手续、路线、交通安排被触发

当然，冲不代表一定顺利，还要看有没有官鬼来压、有没有空破拖后腿。

## 五、不同题目的重点差异

### 1. 问旅行、出差

重点看是否成行、途中是否顺利、外部环境是否配合。

### 2. 问搬家

重点看父母与财爻。父母主合同与手续，财爻常主房屋成本、租售问题、物资安排。

### 3. 问岗位调动

要兼看官鬼与父母。官鬼主岗位与组织安排，父母主通知、流程、批文。

### 4. 问异地发展

不能只看一时成行，还要看应爻与后续承接，判断是否“去了能站住”。

## 六、远近与内外怎么判断

出行卦里，内外卦与应爻常常非常有用。

### 一般规律

- 内卦：近、熟悉、本地、内部环境
- 外卦：远、异地、外部平台、外部机会

### 应爻的位置感

应爻旺而居外，常表示外部环境更强、更主导；世弱应强时，多半是“自己要适应外面”，不是“外面来适应自己”。

## 七、风险怎么看

出行卦中的风险，多看官鬼、白虎、玄武。

### 常见提示

- 官鬼旺：规则压力、阻碍、政策限制、检查、延误
- 白虎：交通险、磕碰、急性问题、突发状况
- 玄武：路线错误、信息不明、隐性拖延、失误、被骗

若官鬼虽旺，但子孙或父母有力，往往表示有阻但可过；若官鬼重克世，又缺少承接，就更应保守。

## 八、应期怎么抓

出行卦的应期，往往很适合抓“动点”。

### 常见抓手

1. 世爻逢冲逢值
2. 父母爻出空、逢值
3. 应爻发动，外部条件到位
4. 合绊解开，行程推进

### 特别提醒

成行时间、到达时间、稳定时间往往不同。尤其是搬迁和异地发展，更要区分“开始动”与“真正稳定”。

## 九、常见误区

### 1. 见冲就怕

在出行迁移卦里，冲常常是成行动力。

### 2. 只看世爻，不看父母爻

很多事情不是人不愿意去，而是手续不通、通知未下、证件没好。

### 3. 外卦强就断一定好

外卦强只说明外部世界在主导，不代表一定适合自己。

### 4. 不区分短途出行与长期迁移

短期看成行与顺利，长期还要看承接与落地。

## 十、实务分析模板

这类卦建议先写四句：

1. 这次能不能真正动起来
2. 卡点在自己、手续还是外部环境
3. 风险是短期波动还是结构性阻碍
4. 动了之后是顺利落地，还是还会继续折腾
