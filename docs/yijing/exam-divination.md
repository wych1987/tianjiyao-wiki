# 六爻考试升学占断

六爻占考试、升学、证书、面试与录取，常常比一般人想象得更复杂。因为这类问题往往不只问“能不能过”，还同时包含准备状态、发挥情况、流程文书、排名名额、通知时间和最终录取几个层次。若只盯一个父母爻或官鬼爻，很容易断得片面。

真正专业的考试类占断，应当把父母、官鬼、世爻、应爻与应期逻辑串起来看。

## 一、考试类问题到底在看什么

考试、升学、证件和录取类问题，通常包含以下几层：

1. 本人准备和发挥状态
2. 试卷、文书、资格和流程
3. 名额、成绩、位次和结果
4. 对方机构、学校或评审方态度
5. 什么时候出结果、什么时候落实

### 专业提醒

“考得怎么样”和“能不能录取”并不总是一回事，要分层判断。

## 二、考试类占断的标准取用顺序

建议按以下顺序取象：

1. 世爻看本人状态与承接力
2. 父母爻看试卷、文书、证件、资格、成绩单和通知
3. 官鬼爻看名次、录取、职位或资格中的位阶因素
4. 应爻看学校、单位、评审方或流程对接面
5. 子孙爻可辅助看发挥、思路、临场释放与缓压能力

## 三、第一层：世爻先看本人能不能接住这件事

考试类问题最容易忽略世爻。很多人只看父母旺不旺，却不看世爻能不能承接。

### 要看的重点

1. 世爻是否得月日扶
2. 世爻是否被官鬼、兄弟或忌神压住
3. 世爻发动是主动争取，还是心态不稳
4. 世爻是否空破，表示本人状态飘、虚、未定

若世爻自身不稳，即使流程不错，也常见发挥失常、信心不足或后续跟进能力差。

## 四、第二层：父母爻是考试类问题的核心用神

父母爻在这类问题里非常关键，因为它主文书、试卷、学业、资格、材料和结果通知。

### 常见判断方向

1. 父母旺相：文书流程、成绩信息、资格条件较有基础
2. 父母空破：消息延迟、材料反复、通知不稳、成绩未定
3. 父母发动生世：有利于考试、录取、消息到来
4. 父母受克：流程卡顿、发挥受阻、结果不如预期

### 专业提醒

考试卦里父母爻往往比官鬼更基础，因为没有“文书与资格”，就谈不上最后的“位次与结果”。

## 五、第三层：官鬼爻决定能不能真正拿到名额与结果

官鬼在考试类占断里，不是单纯坏爻，而常常代表结果中的“位阶”因素，例如录取、编制、资格通过、名次进入等。

### 常见理解

1. 官鬼旺而有情：名额、录取、通过结果更有希望
2. 官鬼旺而克世：结果有压力，竞争激烈，本人未必承得住
3. 官鬼空破：名额不稳、结果悬、消息未定，或有名无实
4. 官鬼与父母相生：流程和结果衔接较好

## 六、第四层：应爻代表学校、单位与评审接口

考试录取不是只看自己，也要看对方那边有没有接纳意愿。

### 应爻常见象义

1. 应生世：对方倾向接纳，接口顺
2. 应克世：流程压力、竞争对手强，或对方标准高
3. 应空：对方信息未定、流程拖延、结果未明
4. 世应相合：更利对接与落实

## 七、第五层：应期要分消息应和结果应

考试类问题特别容易把“查分时间”“通知时间”“最终入学或入职时间”混成一个点。

### 建议分开判断

1. 查分、消息、通知：多看父母爻与应期引发
2. 录取、拿证、定局：多看官鬼与世爻能否承接
3. 入学、报到、正式开始：再看后续流程是否稳定

这样推期会比硬断某一天更稳。

## 八、几类常见考试结构

### 1. 父母旺生世，官鬼有情

多主准备、流程、结果三者都较顺，是比较理想的考试卦。

### 2. 父母旺而世弱

条件和流程不差，但本人状态承接不足，容易发挥打折。

### 3. 世旺而父母空破

自己努力很足，但流程、材料、信息、通知层面容易反复。

### 4. 官鬼旺而克世

竞争很强、门槛很高，事情不是没机会，但压力也大。

## 九、考试类高频问题拆分

### 1. 问能不能考上

重点看父母、官鬼、世爻三者关系。

### 2. 问面试是否通过

除父母、官鬼外，更要重视应爻和世应接口。

### 3. 问证书能不能拿到

父母爻更关键，官鬼辅助看结果中的资格落定。

### 4. 问录取通知什么时候来

应期层要重看父母爻出空、填实、逢值或被引动。

## 十、常见误区

### 1. 只看父母，不看世爻

考试不只是试卷问题，也是本人状态问题。

### 2. 只看通过，不看流程层级

消息应和结果应常常不是同一个时间点。

### 3. 官鬼一出现就当压力

在考试卦里，官鬼很多时候也代表名额、位次、资格结果。

### 4. 不看应爻

对方机构的接口和态度，常常是最终落地的关键。

## 十一、实务分析模板

占考试升学时先答：

1. 本人状态稳不稳
2. 父母爻是否有力
3. 官鬼能否成局并真正落位
4. 应爻是否愿意接纳
5. 消息应、结果应、落实应分别在什么时间层
