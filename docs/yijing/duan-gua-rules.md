# 六爻断卦核心法则

六爻真正难的不是起卦，而是判断时如何抓主线。专业断卦通常都遵循一个固定顺序：先定用神，再看旺衰，再查动变，最后推事件性质与应期。顺序一乱，判断就容易被枝节带偏。

## 一、断卦的基本顺序

建议固定采用以下流程：

1. 明确题目
2. 取用神
3. 看世应
4. 辨日月旺衰
5. 看动爻、变爻、合冲刑害
6. 查旬空、月破、墓绝、进退
7. 综合判断成败、过程与应期

这个流程的核心，是先抓“主轴”，再看“细节”。

## 二、取用神是断卦第一步

### 常见取用原则

| 占题 | 常用用神 |
|------|----------|
| 求财 | 妻财 |
| 工作、升职 | 官鬼 |
| 考试、合同、证件 | 父母 |
| 婚恋男问女方 | 妻财 |
| 婚恋女问男方 | 官鬼 |
| 疾病 | 官鬼为病，子孙为药 |
| 子女 | 子孙 |
| 合作伙伴 | 应爻兼看相关六亲 |
| 失物 | 妻财或父母，视物类而定 |

### 专业提醒

1. 世爻不等于用神。
2. 同一问题可能需要“主用神 + 辅助用神”。
3. 占结果与占过程时，用神焦点可能不同。

例如占求职，官鬼爻主职位本身，世爻主自己状态，父母爻主简历合同，三者都可能需要一起看。

## 三、旺衰判断是六爻判断力的核心

判断旺衰时，至少要看四层力量：

1. 月建
2. 日辰
3. 动爻变爻
4. 其他爻的生扶克泄

### 月建与日辰的权重

- 月建主一时之大环境，力量最稳。
- 日辰主眼前触发点，力量更直接。
- 动爻能主动发力，静爻更依赖日月。

### 旺衰的实际判断

- 得月建、生扶多者，通常为旺。
- 受月破、入墓、逢绝、被重克者，通常为衰。
- 动而化回头生，力量增强。
- 动而化回头克、化绝、化墓，力量下降。

## 四、旬空、月破、日破的真实意义

### 旬空

旬空表示该爻当前“有象未实”或“暂时落空”。它不是绝对无效，而是未落地、未兑现、未归位。

常见断法：

1. 用神空，事情暂时无着落。
2. 忌神空，眼前压力未必真成。
3. 动而出空，往往就是事情启动之时。

### 月破

月破属于大环境不支持。用神月破，主根基不稳，外部条件不足。若同时得日扶、动化回生，也可能“破而后立”，但绝不宜轻断为吉。

### 日破

日破更多表现为当下受冲、当前失力、眼前反复。比月破轻，但在短期占断中影响很明显。

## 五、动爻、变爻与回头作用

断卦不能只看“动不动”，还要看“动成什么”。

### 回头生

动爻化出之爻反过来生扶本爻，主后续转强、获得支持、事情有补救空间。

### 回头克

动爻化出之爻反而克制本爻，主自损、自我消耗、后劲不足。

### 化进神与化退神

- 化进神：事情向前推进，趋势增强。
- 化退神：事情退缩、延后、缩减。

专业断法里，很多“先吉后凶”或“先难后成”的判断，都来自回头作用与进退神的组合。

## 六、冲合刑害如何落到具体事件

### 合

合不一定是好，重点看是“绊住”还是“成就”。

- 用神得合而成局，可能主合作成事。
- 用神被合住不动，也可能主拖延、被牵制。

### 冲

冲表示动、散、分离、脱离、触发。

- 占出行，冲可能是成行。
- 占婚恋，冲可能是争执或分离。
- 占疾病，冲可能是病情激发或治疗介入。

### 刑害

刑害多属不顺、暗伤、别扭、隐性消耗，往往不像冲那样外显，但在感情、职场、合同类问题中很常见。

## 七、按题目判断的几个抓手

### 求财

1. 财爻是否旺相
2. 世爻能否承财
3. 兄弟是否夺财
4. 官鬼是否形成规则或成本压力

### 婚恋

1. 男问财、女问官
2. 世应是否有生合互动
3. 用神是否受冲破空亡
4. 兄弟、子孙是否构成旁扰

### 工作与升职

1. 官鬼是否得令
2. 世爻是否能承职位压力
3. 父母爻是否主文书录用流程顺畅
4. 应爻是否代表单位真实意向

### 疾病

1. 官鬼看病邪
2. 子孙看药与恢复力
3. 世爻看身体承受度
4. 白虎、玄武、腾蛇常补充病象细节

## 八、应期怎么推

应期不能脱离事件性质去单独推。先判断“会不会发生”，再判断“什么时候发生”。

### 常见应期方法

1. 用神逢值
2. 用神逢合而成
3. 用神逢冲而动
4. 空爻出空
5. 破爻逢补
6. 墓爻逢冲墓

### 短期和长期的区别

- 短期问事，多以日时为主。
- 中期问事，多兼看月令。
- 长期问事，需结合大环境变化，不宜死扣一日一时。

## 九、容易导致误判的几个问题

### 1. 只看一个用神

复杂题目常有多层结构，例如求职既要看官鬼，也要看世爻承受力和父母爻手续。

### 2. 把空亡当绝对没有

空亡更常见的是“暂不到位”，不是完全不存在。

### 3. 看到合就断成，看到冲就断坏

合与冲都必须结合题目语境。职业调动、搬家、出行，很多时候就是“冲则成”。

### 4. 忽略世应关系

很多感情卦、合作卦、交易卦，问题不在财官旺不旺，而在双方有没有真实互动。

## 十、实战训练建议

建议把每个案例都拆成四句：

1. 用神是谁
2. 现在旺还是衰
3. 阻力来自哪里
4. 何时触发变化

如果这四句能稳定写清，六爻断卦能力就会明显提升。
