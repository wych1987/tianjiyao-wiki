# 六爻失物寻人与寻宠占断

失物、寻人、寻宠这类卦，和婚姻、求财、事业类问题不同，它的重点不只是“成不成”，更在于“东西还在不在”“人有没有动”“距离远近如何”“方向落在哪里”“最终能否回归”。

传统六爻在这类问题上，优势很明显，因为卦爻本身带有内外、远近、动静、显隐的信息。《卜筮正宗》《增删卜易》对于失物、行人、逃亡、家宅失脱都有可参照的规则，落实到现代知识库写法里，核心仍是四件事：

1. 定对象
2. 看存亡
3. 看动静
4. 看归期与方向

## 一、不同问题的主用神

### 1. 丢失钱财、货物、贵重物品

一般以妻财爻为主。财爻代表失去之物，也代表物品最终是否还能回到自己手中。

### 2. 证件、合同、票据、文件

多以父母爻为主。父母在六爻中本就主文书、纸面信息、凭证、手续类事物。

### 3. 宠物、孩子、学生、晚辈

多参考子孙爻。子孙不仅主子女，也主宠物、小辈、依附于自己的弱势对象。

### 4. 寻找成年人、联系人、对方去向

除了世应，还要根据关系选六亲。若是问伴侣、亲友、合作对象，往往需要把应爻与对应六亲一起看。

## 二、失物卦的判断主轴

失物类问题最核心的判断主轴通常有四个：

1. 用神是否仍在卦中显现
2. 用神旺衰如何
3. 用神是静是动
4. 世与用神之间是否还有回归关系

### 常见结构理解

- 用神旺而静：物仍在，未远离，多半停在原处或近处
- 用神旺而动：物有流转、转手、被移动的象
- 用神空破：东西未必彻底无，但当下难见、难得、信息模糊
- 用神伏藏：失物不显，多在遮蔽、夹层、柜内、袋中、隐蔽处

## 三、世爻与用神的关系

世爻在失物卦里代表问卦人自己，既代表“寻找者”，也代表“与失物的连接能力”。

### 常见关系

1. 世生用：自己主动找、主动追、主动取回
2. 用生世：失物仍有归主之机，容易找回
3. 世克用：自己控制欲强，但可能找得急躁
4. 用克世：失物相关问题反过来压制自己，常见焦虑、找不到线头

若世与用神完全无情，通常代表失而离散、线索弱、找回难度高。

## 四、动静与远近判断

### 1. 静爻

多主原地、近处、未离开、尚未被转移。

### 2. 动爻

多主移动、转手、被他人带走、事态仍在发展中。

### 3. 内外卦

- 内卦多主近处、家中、身边、熟悉环境
- 外卦多主外部、远处、别人处、非本位空间

### 4. 应爻

应爻若强，多表示东西、人、线索落在外部环境或他人手里。

## 五、方位怎么判断

六爻方位判断，通常要结合卦宫、内外卦、地支方位与六神象义综合判断。

### 八卦常见方位

- 乾：西北
- 坤：西南
- 震：东方
- 巽：东南
- 坎：北方
- 离：南方
- 艮：东北
- 兑：西方

### 地支常见方位

- 亥子丑偏北
- 寅卯辰偏东
- 巳午未偏南
- 申酉戌偏西

### 实务原则

方位不宜只用单一点判断，应以“本卦方位 + 用神地支方位 + 动静内外”交叉验证。

## 六、寻人的判断重点

寻人卦比失物卦更复杂，因为人有主观行动能力。

### 重点要看

1. 应爻是否发动
2. 所取用神是否旺相
3. 内外卦显示其在近处还是远处
4. 官鬼是否表示阻碍、意外、风险、躲避
5. 父母是否表示消息、联系、文书记录

### 常见象义

- 应动外卦：人已外出或在外部环境中活动
- 用神伏藏：人有躲避、隐匿、不愿现身之象
- 父母发动：有消息、电话、记录、文书线索
- 官鬼旺：可能有风险、压力、受控、出事、受阻

## 七、寻宠物的特别处理

寻宠通常以子孙为主，但要特别结合腾蛇、玄武、白虎等信息。

### 常见判断

- 子孙旺静：宠物尚在附近
- 子孙动外卦：宠物已跑远或被带离
- 玄武临子孙：可能躲藏、钻角落、入暗处
- 腾蛇临子孙：惊吓乱窜、方向反复
- 白虎临子孙：要留意受伤、惊险、外力冲击

## 八、能不能找回

找回与否，关键看用神有没有回归关系。

### 有利找回的常见信号

1. 用神生世
2. 用神与世相合
3. 用神虽空但能出空
4. 用神伏而可引出
5. 父母或应爻提供明确线索

### 不利找回的常见信号

1. 用神受重克且无救
2. 用神动而化绝、化破、化退
3. 世与用神完全无情
4. 用神已远离外卦且被他爻合住

## 九、应期判断

失物寻人卦的应期，一般抓以下节点：

1. 用神逢值
2. 用神出空
3. 动爻所化之期
4. 父母爻发动对应消息到来
5. 冲开合绊之时

### 实务提醒

找回时间不一定等于发现时间。有时先得消息，后取回；有时先定位，后处理。

## 十、常见误区

### 1. 丢什么都一律看财爻

证件、文件、合同类更应重父母；宠物、孩子等应重子孙。

### 2. 见动就断找不到

动只是表示变化与移动，不必然表示彻底丢失。

### 3. 方位只看卦名

真正专业的方位判断必须结合卦宫、地支、内外卦与六神象义。

### 4. 寻人卦忽略应爻

人在卦里有主观能动性，应爻和动静远比静态六亲更重要。

## 十一、实务分析模板

这类卦建议先写四句：

1. 对象现在还在不在、近不近
2. 它是静止还是仍在移动
3. 线索落在自己手里还是外部环境里
4. 最可能先出现的是消息、定位还是直接回归
