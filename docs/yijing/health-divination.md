# 六爻疾病健康占断

疾病卦是六爻中最需要克制和边界感的一类。它可以用于判断趋势、风险、轻重缓急与恢复节奏，但不能替代现代医学检查与诊疗。《卜筮正宗》《增删卜易》虽均有病症占例，但真正用在现代知识库写作中，必须把“提示”和“诊断”明确区分。

专业疾病卦的核心，不是神秘化病名，而是判断四件事：

1. 病邪强不强
2. 人体承不承得住
3. 有没有解法
4. 变化会往好还是往坏走

## 一、疾病卦的三条主线

疾病类问题最核心的三条线是：

1. 官鬼
2. 世爻
3. 子孙

### 三者分工

- 官鬼：病邪、病因、病灶、压力源
- 世爻：命主体质、当下承受力、整体状态
- 子孙：医药、治疗、恢复、排病与缓解能力

必要时还要看父母爻，因为父母可能代表检查单、病历、医院、药方、文书结果。

## 二、官鬼怎么看

### 1. 官鬼旺相

表示病势或压力源较强，问题不轻，需重视。

### 2. 官鬼空破

常见于：

- 病象未明
- 检查反复
- 有症状但未确诊
- 压力感明显，但病理未必稳定成形

### 3. 官鬼克世

病邪直接压制身体承受度，多主症状明显、压力偏重、恢复慢。

### 4. 官鬼受制

若子孙发动制鬼，或官鬼受日月制约，多表示病情可控、治疗有路、风险正在下降。

## 三、世爻决定身体承受度

疾病卦里，很多人只看官鬼，不看世爻，这是最大的误区之一。因为同样的病邪，落在不同体质与阶段上，结果完全不同。

### 世爻常见判断

1. 世旺：体质尚可，恢复弹性较好
2. 世弱：承受力不足，恢复慢，容易反复
3. 世受生：有救、有支撑
4. 世受克：体感强，恢复压力大

若世爻虽弱，但有日月扶、动爻护，也不能轻判凶；反过来，世爻表面不差但官鬼极旺、白虎并临，也要提高警惕。

## 四、子孙是疾病卦中的关键“解法爻”

子孙在病卦中常主医药、治疗方案、恢复力、缓解机制。

### 常见结构

- 子孙制鬼：治疗有效，病势可缓
- 子孙生世：恢复力增加，身体承受更好
- 子孙空破：药效未显、方法不对、恢复慢
- 子孙伏藏：解法暂未显形，可能还没找到正确路径

## 五、六神在疾病卦中的补充信息

六神不能替代主体判断，但在病卦里常能补出细节风格。

### 常见提示

- 白虎：外伤、炎症、手术、疼痛、血光类风险
- 玄武：隐症、慢性问题、反复、难言之疾、拖延
- 腾蛇：焦虑、疑病、反复不定、神经性或情绪性干扰
- 朱雀：咽喉、口舌、炎热、信息结果、报告反馈

这些都只能作辅助参考，不能脱离官鬼、世爻、子孙单独断病。

## 六、不同题目的断法重点

### 1. 问有没有病、重不重

重点看官鬼与世爻的对抗程度，以及官鬼是否得令得气。

### 2. 问检查结果

要兼看父母爻，因为父母主检查、报告、文书结果。官鬼空而父母动，常见“正在查、尚未定论”。

### 3. 问治疗有没有效

重点看子孙能否制鬼、生世，及官鬼是否被削弱。

### 4. 问手术

需特别留意：

- 白虎信息
- 官鬼旺衰
- 子孙是否有效制鬼
- 世爻是否承受得住

手术卦不能只看凶象，还要看“做不做更合适”。有时官鬼太重，反而需要白虎之象来“破病出路”。

## 七、恢复与应期判断

病卦的应期，往往不是疾病完全结束的时间，而是：

1. 症状转折点
2. 检查明确点
3. 治疗起效点
4. 风险减轻点

### 常见抓手

- 子孙逢值
- 官鬼受冲受制
- 世爻得扶出空
- 父母逢值，报告结果明朗

## 八、风险边界与专业写作原则

疾病占断的专业性，不是把病说得多具体，而是知道哪些能说、哪些不能说。

### 必须坚持的边界

1. 六爻可辅助判断趋势，不替代医学诊断
2. 重大病症、手术、长期治疗，优先遵从医院检查和医生意见
3. 卦象更适合判断轻重、节奏、变化点与恢复条件

这既是现代知识库应有的审慎态度，也是对传统术数合理定位。

## 九、常见误区

### 1. 见官鬼就断大病

官鬼在病卦中是必看的，但其强弱、是否克世、是否被制，决定问题层级。

### 2. 只看病邪，不看体质

世爻代表身体承受度，不看世爻，就很难判断轻重与恢复节奏。

### 3. 把白虎当绝对凶

白虎在手术卦里有时是必要之刀，不一定纯坏。

### 4. 用卦替代检查

这是最危险的误区。卦只能提示方向，不能代替现代诊疗工具。

## 十、实务分析模板

病卦建议先写四句：

1. 病邪是否明确且有力
2. 身体承受度如何
3. 有没有明确治疗或缓解路径
4. 变化会先出现在检查、治疗还是恢复层面
