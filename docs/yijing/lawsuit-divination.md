# 六爻官司诉讼占断

官司诉讼类问题，在六爻里本质上属于“对抗结构卦”。它不只是看谁对谁错，更看谁占势、谁有证据、谁能承压、谁最终更容易拿到结果。现代场景下，这类卦还常常包括仲裁、投诉、合同争议、平台申诉、劳动纠纷、税务争议等。

这类卦判断时最重要的，不是情绪化站队，而是抓四条主线：

1. 官鬼
2. 父母
3. 世应
4. 朱雀与白虎等辅助信息

## 一、主用神怎么取

### 1. 官鬼

官鬼在此类问题中常主官非、规则压力、裁判力量、诉讼程序本身。

### 2. 父母

父母主证据、文书、合同、记录、申诉材料、判决书、流程文件。

### 3. 世应

世爻代表自己，应爻代表对方。官司卦中，世应的强弱与互动，往往直接反映双方对抗态势。

## 二、为什么父母爻在官司卦里很重

很多初学者只盯官鬼，这是不够的。现代纠纷大多不是纯口头争执，而是证据、材料、记录、合同、程序的比拼。父母爻往往比情绪本身更决定胜负。

### 常见判断

- 父母旺相：证据条件较完整，程序面较稳
- 父母空破：材料不足、手续缺漏、证据链不稳
- 父母生世：证据更利于自己
- 父母生应：材料更偏向对方或规则站在对方那边

## 三、世应对抗怎么看

世应是官司卦的对抗轴。

### 常见结构

1. 世旺应弱：自己更有主动与承压能力
2. 世弱应旺：对方更强势，自己处于被动
3. 世应相冲：冲突公开化，争议正面爆发
4. 世应相克：双方难和，矛盾深
5. 世应相合：可能和解，也可能胶着不下，要分清是和还是绊

## 四、官鬼在诉讼卦里的两层含义

官鬼一方面代表压力、麻烦、程序、裁判力量，另一方面也代表“事情是否正式进入规则系统”。

### 常见理解

- 官鬼旺：程序明确，压力真实，事情已进入较正式阶段
- 官鬼空：程序未定、吓唬成分大、尚未真正立案或落实
- 官鬼生世：规则可能偏向自己，或自己更能借规则发力
- 官鬼克世：规则压力更大，自己需承担更重负担

## 五、朱雀、白虎、玄武的辅助信息

### 朱雀

主口舌、争辩、文书、传播、说辞。诉讼卦中极为常见。

### 白虎

主强硬、损伤、罚责、激烈后果，常提示事情升级、代价加重。

### 玄武

主隐情、欺瞒、拖欠、虚构、暗中操作，对合同纠纷、财务纠纷尤其重要。

### 勾陈

主拖延、旧事缠绕、程序迟滞，常见于长时间扯皮。

## 六、不同纠纷类型的重点

### 1. 合同纠纷

重点看父母爻与财爻。父母主合同文本，财爻主利益标的。

### 2. 劳动仲裁

重点看官鬼、父母、世应，外加财爻判断赔付、工资、补偿。

### 3. 民事诉讼

重点看世应强弱、父母证据链、官鬼程序是否真正推动。

### 4. 平台申诉、投诉、行政处理

重点看父母爻是否能成文书证据、官鬼是否形成正式裁断、应爻是否仍有回旋空间。

## 七、胜负与结果怎么判断

官司卦不宜简单断“赢”或“输”，更准确的是判断：

1. 谁更占势
2. 谁更有证据
3. 是否有和解空间
4. 最终代价落在哪一方更重

### 常见有利信号

1. 世旺且得父母生扶
2. 官鬼有情而不压世
3. 应爻衰弱或受制
4. 父母文书线站在世爻一方

### 常见不利信号

1. 世弱应旺
2. 父母空破无助
3. 官鬼重克世
4. 白虎、朱雀、玄武共同加压而无制

## 八、和解还是硬碰硬

很多官司卦真正要回答的，不是“判不判”，而是“适不适合继续打下去”。

### 判断思路

- 世应相合且父母不破：和解空间较大
- 世应相冲且白虎、朱雀并重：冲突升级概率高
- 官鬼旺而父母弱：事情已大，但自己准备不足
- 父母旺而官鬼未实：更适合先整理材料，不宜急于开战

## 九、应期判断

这类卦的应期通常要分阶段：

1. 何时立案或正式进入程序
2. 何时收到通知、文书、回复
3. 何时进入裁判、和解或执行节点

### 常见抓手

- 官鬼逢值：程序启动
- 父母逢值：文书到来
- 世应逢冲：正面交锋、消息爆出、关系激化
- 合绊解开：和解或判决推进

## 十、常见误区

### 1. 官鬼旺就断自己输

官鬼旺有时只是说明程序真实、规则介入深，不等于必败。

### 2. 不看父母爻

现代纠纷很多输在证据，而不是输在情理。

### 3. 世应相合就断和解

有时是和，有时是互相牵制、久拖不决。

### 4. 只看情绪，不看规则链

诉讼卦的主战场常在程序和材料，不在主观对错感。

## 十一、实务分析模板

官司卦建议先写五句：

1. 事情是否已真正进入规则系统
2. 哪一方当前更占势
3. 证据与文书面偏向谁
4. 是适合和解还是适合推进程序
5. 先来的会是通知、对抗还是结果
