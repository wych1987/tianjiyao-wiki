# 六爻应期判断

六爻断卦里，最能拉开水平差距的地方之一，就是应期。很多人会看吉凶，却不会回答“什么时候应”；或者一味追求精确到某一天，却忽略了事件本身的层级、结构与成熟条件。真正专业的应期判断，不是机械套口诀，而是先判断事情能不能成，再判断它以什么节奏成熟，最后才去看月、日、时如何引发。

所以，应期不是附属技巧，而是断卦闭环的重要组成部分。

## 一、应期判断到底在看什么

六爻推应期，至少要回答以下几个问题：

1. 事情有没有成熟基础
2. 事件更像近应、远应还是阶段应
3. 关键引发点落在动爻、合冲、填实还是出空
4. 月日时谁是主要触发层
5. 应的是结果、消息，还是过程中的某一步

### 专业提醒

很多“应期不准”，不是不会推，而是根本没先分清问的是结果、消息还是动作本身。

## 二、推应期前的基本顺序

建议任何卦先按以下顺序处理：

1. 先判断事情能不能成
2. 再判断主用神当前是旺是衰、动还是静
3. 再看阻碍点来自何处
4. 最后判断什么时间层会把这件事推出来

若事情本身无承接，强行推某天某月，往往只会走偏。

## 三、第一层：动爻是最直接的应期入口

动爻代表变化、进程、引发点，因此很多应期首先从动爻入手。

### 常见判断方向

1. 动爻逢合、逢冲时是否成事
2. 动爻化进、化退代表快慢差异
3. 动爻若化空、化破，常主延迟或过程反复
4. 动爻若直接生扶用神，应事往往较快

### 专业理解

动不是一定马上应，而是说明事件已经进入运行状态。真正何时落地，还要看月日时是否配合。

## 四、第二层：合冲决定事情是启动、定局还是破局

合冲在应期里非常关键，但必须结合事件性质来理解。

### 合的常见意义

1. 关系、事项被牵住
2. 条件正在聚合
3. 需要等合成、合稳，事件才落地

### 冲的常见意义

1. 静局被打破，事件开始动
2. 原有状态解开，事情进入新阶段
3. 某些关系题、出行题、变动题反而需要冲来启动

### 专业提醒

合不一定就是拖，冲也不一定就是坏。关键要看问的是“要不要动”，还是“要不要稳”。

## 五、第三层：旬空、月破、入墓、伏神如何影响时间判断

### 1. 旬空

用神旬空，常见事情未实、消息未定、心态未决。推期时常看出空、填实、逢值。

### 2. 月破

月破多主当下难成、力量不稳，若后续得日扶或待后月转旺，才更易应。

### 3. 入墓

入墓常见事情被压住、收住、不显，推期时常看冲墓、开墓之时。

### 4. 伏神

伏神主未显、未露、未动。推期往往要等伏神得引、得冲、得生时，事情才会浮出水面。

## 六、第四层：月、日、时的不同职责

很多初学者一上来就想断某一天，但实际上月、日、时各有分工。

### 月令

更像阶段背景，决定事情在哪个时间段更有条件成熟。

### 日辰

更像具体触发点，常用来看某个动作、消息、见面、签约、回复等。

### 时辰

通常只在事件已非常接近、问题问得很细、且卦象已经明确时才作为细化层使用。

### 专业原则

越大的事，越先看月和阶段；越小、越近、越具体的事，才逐步收缩到日时。

## 七、几类常见应期判断模式

### 1. 近应型

用神旺相、动而有情、世应承接好，常在近日内见消息、动作或结果。

### 2. 出空型

用神空而未实，常待出空、逢值、填实之后应事。

### 3. 冲开型

事情被合住、墓住、伏住，往往要等冲开后才动。

### 4. 阶段成熟型

当前不具备条件，需要等某月、某阶段整体气候转变，事情才真正落地。

## 八、不同题目的应期重心不同

### 感情题

常看世应、官财、合冲，消息与定局应期要分开判断。

### 求财题

常看财爻何时得生、何时到账、何时结算，钱路和钱到手未必同一时间。

### 事业题

官鬼、父母、应爻都重要，通知时间、入职时间、岗位稳定时间也常不同。

### 疾病题

更适合看症状转折、检查结果、治疗见效和风险窗口，而不是机械追一天。

## 九、常见误区

### 1. 事情还没成，就硬推具体日期

先判成败，后推时间。

### 2. 把所有应期都压到某一天

很多事情本来就分为消息应、动作应、结果应三个层次。

### 3. 只背口诀，不看卦意

应期必须服从事件类型和卦象结构。

### 4. 忽略月令背景

只看日时，常会把阶段问题误读成短期触发问题。

## 十、实务分析模板

推应期时先答：

1. 事情已经成熟了几成
2. 关键卡点落在空、破、墓、伏还是未动
3. 要等的是启动、消息还是定局
4. 应期更应落在月、日还是某个阶段节点
