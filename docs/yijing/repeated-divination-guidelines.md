# 六爻重复问卦与提问规范

很多人不是不会起卦，而是在真正发问之前，没有把问题收敛好。

最常见的情况有三种：

1. 同一件事连续换措辞追问
2. 结果还没落地，就因为焦虑立刻再起一卦
3. 把“能不能成、对方怎么想、什么时候有消息、最后会不会成”混成一个问题

六爻适合处理具体事件，但它要求问题有边界。你问得越散，卦就越容易被情绪带偏；你问得越清，判断顺序才越容易稳定。

这页不是教你怎么断卦，而是帮你在起卦前先把问题问对。

## 一、为什么六爻强调“一事一卦”

六爻之所以适合现实决策，不是因为它能替你回答所有问题，而是因为它能对一件具体事情的当前阶段给出结构判断。

所谓“一事一卦”，重点不在形式，而在边界：

1. 一个问题最好只对应一件具体事情
2. 一个问题最好只对应一个主要判断目标
3. 一个时间窗口里，先判断当前阶段，再谈下一阶段

比如感情题里，“对方最近会不会主动联系我”和“这段关系最终能不能复合”并不是同一层。前者更像眼前接口，后者更像关系结果。如果一开始把它们压成同一个问题，后面取用神、分主次和推应期都容易乱。

## 二、什么算“同一个问题”

判断是不是同题，重点不在措辞像不像，而在下面四件事有没有变：

1. 对象是不是同一个人、同一个岗位、同一个项目或同一件具体事
2. 决策点是不是同一个，比如都是在问“接不接这个 offer”
3. 时间窗口是不是同一阶段，比如都在问“本周内有没有消息”
4. 现实里有没有出现新的动作、通知、回复、合同或阶段变化

如果这四件事都没变，只是从“他会不会来找我”换成“他最近会不会主动联系我”，本质上大概率还是同题。

相反，下面这些情况通常已经不算同一问题：

1. 上次问的是“能否收到面试反馈”，这次问的是“已经收到 offer 后要不要接”
2. 上次问的是“对方近期会不会恢复联系”，这次是“已经恢复联系后，关系能不能进入稳定推进”
3. 上次问的是“这个项目能不能启动”，这次是“项目已启动后，未来一个月能不能回款”

六爻更看“事件有没有进入新阶段”，而不是只看你有没有换一句话。

## 三、什么时候不应该重复问卦

以下几种情况，通常都不建议立刻再问：

1. 事情本身还没有出现新变量
2. 你只是因为上一卦结果不符合预期
3. 你把同一个结论拆成多个情绪化追问
4. 你还没等到消息应、动作应或结果应出现

尤其是第四种最常见。很多人今天问完，明天没动静，就怀疑卦不准，然后再问一次。但现实事件本来就分层，有些先应在回复，有些先应在见面，有些先应在真正结果。没等事件走到应点，就急着重问，很容易把注意力从事件本身拉回情绪本身。

如果你对时间判断拿不准，优先接着读：

1. [六爻应期判断](response-timing.md)
2. [六爻应期案例专题](response-timing-cases.md)

## 四、什么时候可以重新起卦

能不能再问，没有统一的“隔几天就可以”规则。核心不是天数，而是事件是否进入新阶段。

更适合重新起卦的情况通常有：

1. 现实里出现了实质性新变量，比如对方已回复、HR 发来新方案、合同条款变了、检查结果出来了
2. 上一个问题已经结束，现在进入下一决策点
3. 原问题太大太散，现在你终于能把它收敛成一个具体问题

也就是说：

1. 没有新变量，隔再久也可能还是旧题
2. 有了新变量，即便时间不长，也可能已经是新题

六爻不是靠频繁重问提高准确度，而是靠在正确阶段问正确问题。

## 五、起卦前怎么把问题问清楚

一个更适合六爻的问题，通常至少包含四个要素：

1. 主体：谁在问，涉及谁
2. 事件：到底是哪件具体事
3. 目标：你要判断的是能否推进、对方态度、是否录取，还是何时有消息
4. 时间：你关心的是这周、这个月，还是这一轮流程

可以把问题尽量收敛成下面这种形式：

1. 未来 30 天内，对方是否会主动恢复联系
2. 这次正在谈的 A offer 是否值得接受
3. 这个副业项目在未来 3 个月内是否有实际回款可能
4. 本轮诉讼在当前阶段是否会出现有利进展

相对不理想的问题通常是：

1. 我们还有希望吗
2. 我财运怎么样
3. 我工作会不会变好
4. 他到底怎么想我

这些问法不是完全不能用，而是过于宽泛，容易把关系结构、事件推进、时间判断和情绪确认混在一起。

## 六、同一件事，应该怎么拆问题

面对复杂现实，真正有效的做法不是一卦包打天下，而是先找当前最关键的决策点。

例如感情问题：

1. 先问：未来 30 天内，对方会不会恢复联系
2. 如果恢复联系，再问：这段关系能不能进入稳定推进
3. 如果已经进入互动，再问：更可能在哪个阶段见分晓

例如求职问题：

1. 先问：这次流程能不能拿到明确结果
2. 如果结果已出，再问：这个 offer 值不值得接
3. 如果已经入职，再问：短期内是否适合继续留下

把问题按事件阶段拆开，通常比反复问同一句话更有信息量。

## 七、结果不满意，或者觉得“不准”时怎么办

先不要急着再起一卦，优先检查下面几件事：

1. 你是不是把多个层次的问题混在一起问了
2. 你是不是把消息应、动作应和结果应看成了同一个应点
3. 现实里是不是还没出现足够的新变量
4. 这件事本身是不是更适合先看长期结构，而不是只看眼前事件

如果你问的是长期关系底盘、职业长期方向、财富结构选择，而不是眼前单一事件，直接反复用六爻追问，往往不如回到八字或紫微看底盘更稳。可以先用：

1. [体系选择指南：什么问题该用八字、紫微还是六爻](../practice/system-selection-guide.md)

## 八、去主站前的最小检查清单

如果你准备去主站真实起卦，先快速过一遍这 5 个问题：

1. 我问的是不是一件具体事情，而不是一团情绪
2. 我这次最想判断的是结果、态度、时机还是接口
3. 对象和时间范围是不是明确
4. 我是不是只是因为不接受上一次结果，才想再问一次
5. 如果这是长期结构问题，我是否应该先切到八字或紫微

这五项里只要有两三项还说不清，先别急着起卦，先把问题重新收敛。
