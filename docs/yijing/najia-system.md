# 六爻纳甲装卦体系

六爻预测真正进入专业层面，关键不在会不会起卦，而在能否把卦装对。纳甲、六亲、六神、世应、伏神，是六爻断事的共同底盘。装卦一旦偏差，后续旺衰、取用、应期几乎都会连带失真。

## 一、什么是纳甲装卦

纳甲装卦，是把六十四卦重新放回干支五行系统中，使卦爻不再只是阴阳符号，而成为可以判断旺衰、生克、冲合、刑害的人事结构。专业六爻断卦通常都以纳甲体系为基础。

装卦至少包含五个层次：

1. 定卦宫
2. 纳地支
3. 配六亲
4. 安六神
5. 定世应并检查伏神

## 二、八宫与纳甲的理解框架

六十四卦分属八宫，八宫既是归类方式，也是六亲安配的参照基准。判断六亲时，不是直接看问事对象，而是先看“本宫五行是什么”。

### 八宫五行

| 宫 | 五行 | 核心气质 |
|----|------|----------|
| 乾宫 | 金 | 刚健、主导 |
| 兑宫 | 金 | 言语、悦纳 |
| 离宫 | 火 | 文明、显露 |
| 震宫 | 木 | 发动、进取 |
| 巽宫 | 木 | 渗透、经营 |
| 坎宫 | 水 | 险阻、流动 |
| 艮宫 | 土 | 停止、守成 |
| 坤宫 | 土 | 包容、承载 |

先确定卦属哪一宫，再看各爻地支与本宫五行之间的关系，才能安六亲。

## 三、六亲安配的核心逻辑

六亲不是现实中的固定亲属，而是五行关系的命名方式。以“宫五行”为我，再看各爻五行与我的生克关系：

| 五行关系 | 六亲 | 常见象义 |
|----------|------|----------|
| 同我 | 兄弟 | 竞争、同类、同伴、分财 |
| 生我 | 父母 | 文书、房屋、学历、长辈、保护 |
| 我生 | 子孙 | 技能、成果、福气、下属、医药 |
| 我克 | 妻财 | 财物、资源、项目、客户、异性 |
| 克我 | 官鬼 | 压力、工作、制度、疾病、忧患 |

### 六亲不是一成不变的“吉凶标签”

- 妻财不只代表钱，也可能代表订单、项目、客户。
- 官鬼不只代表凶，也可能代表岗位、责任、约束力。
- 子孙不只代表子女，也常代表方案、释放压力、解决问题的能力。

断卦时要先判断“所占何事”，再决定哪一类象义最贴切。

## 四、六神的作用层级

六神不决定卦的主体成败，但会补充事件风格、过程气质与细节信息。

| 六神 | 常见信息 |
|------|----------|
| 青龙 | 喜庆、贵助、体面、人情、顺势 |
| 朱雀 | 言语、文书、传播、争论、消息 |
| 勾陈 | 拖延、土地产权、旧事缠绕 |
| 腾蛇 | 反复、疑虑、虚惊、幻想、不实 |
| 白虎 | 风险、损伤、压力、手术、刚性冲突 |
| 玄武 | 隐情、欺瞒、拖欠、暗中操作 |

### 使用原则

1. 先看用神旺衰，再看六神修饰。
2. 六神更适合补充事件风格，不宜脱离五行单独断事。
3. 同一六神在不同题目中含义不同，必须结合问事场景。

## 五、世应是断卦的坐标轴

世爻代表求测者一方，应爻代表对方或外部环境。很多断卦失准，不是不会看生克，而是没有先站稳世应关系。

### 世应的基础用法

1. 世旺应衰，主自己更有主动权。
2. 世衰应旺，主自己受外部条件牵制。
3. 世应相生，易有合作、回应、接洽。
4. 世应相克，易有对抗、误解、阻碍。
5. 世动应静，多为自己主动推动。
6. 应动世静，多为对方或外界先起变化。

### 什么时候不用“世爻就是我”简单化处理

占疾病、失物、官司、胎产等题目时，世爻仍代表求测者立场，但用神未必取世。专业断法必须把“世应”和“用神”分开处理。

## 六、伏神与飞神怎么看

伏神常出现在本卦中用神不显、信息被压住的时候。伏神不是一定无力，而是“有事未透”。

### 判断要点

1. 先看伏神是否为所求用神。
2. 再看飞神是否压制伏神，还是反而护持伏神。
3. 伏神得日月扶起、动爻引出、冲空填实时，往往就是事情显化的时候。

### 常见象

- 财伏官下：钱被制度、手续、压力压住。
- 官伏财下：责任因现实利益而起。
- 用神伏而受制：事未现形，或有隐情未明。

## 七、专业装卦流程

建议把装卦固定成以下流程：

1. 写出本卦、变卦与动爻
2. 确定卦宫
3. 纳入六爻地支
4. 按宫五行安六亲
5. 按日干起六神
6. 确定世应位置
7. 查空亡、月破、日破
8. 如有缺用神，再查伏神

这个流程的价值，不在于形式整齐，而在于避免“边看边猜”的低级错误。

## 八、初学者最常犯的装卦错误

### 1. 直接按现实关系安六亲

比如占父亲，就把父母爻当作“唯一重点”，而忽略题目真正要看的是病情、结果、归来时间还是费用负担。

### 2. 只记表格，不理解宫五行

六亲安配的底层逻辑是宫五行，不理解这一点，装卦只能停留在机械记忆。

### 3. 六神喧宾夺主

看到白虎就判凶，看到青龙就判吉，是最常见的误区。六神只能辅助，不能替代旺衰生克。

### 4. 忽视世应位置

很多合作、感情、交易类问题，关键不是“用神有没有”，而是世应有没有形成真实互动。

## 九、学习建议

如果想把六爻从兴趣提升到专业能力，纳甲装卦必须练到“看卦即能快速还原结构”。建议训练方式如下：

1. 每天手工装 3 到 5 个卦
2. 每次都写清卦宫、六亲、六神、世应
3. 先不急于断吉凶，先保证结构不出错
4. 用真实案例反复核对用神与伏神的落点
