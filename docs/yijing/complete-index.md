## 七、按阅读目标进入

### 1. 想快速建立六爻主线

建议顺序：

1. [六爻占卜入门](liuyao.md)
2. [六爻纳甲装卦体系](najia-system.md)
3. [六爻断卦核心法则](duan-gua-rules.md)

5. [六爻应期判断](response-timing.md)
6. [六爻与周易常见问题](faq.md)

### 2. 想直接解决具体问题

1. 起卦前先读 [六爻重复问卦与提问规范](repeated-divination-guidelines.md)，先把问题收敛成一件事
2. 关系题先看 [六爻婚姻感情占断](marriage-divination.md)
3. 财富事业题先看 [六爻求财与投资占断](wealth-divination.md) 和 [六爻事业与求职占断](career-divination.md)
4. 其他现实题可直接进考试、官司、失物、出行这些分类页

### 3. 想练时间判断

先读 [六爻应期判断](response-timing.md)，再用 [六爻应期案例专题](response-timing-cases.md) 做校准，会比直接背日期规则更稳。
