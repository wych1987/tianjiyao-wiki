## 🔧 部署说明

### GitHub Pages部署
1. **确保设置正确**
   - 仓库设置 → Pages → Source 选择 `main` 分支的 `/docs` 目录
   - 自定义域名已配置：`wiki.tianjiyao.com`

2. **DNS配置检查**
   - 确保 `wiki.tianjiyao.com` 解析到 GitHub Pages
   - 启用HTTPS（GitHub Pages自动配置）

### Google Analytics配置
```yaml
# 在 _config.yml 中替换以下配置
google_analytics: "您的GA4测量ID"  # 格式：G-XXXXXXXXXX
```

### Google Search Console
1. 添加域名 `wiki.tianjiyao.com`
2. 提交sitemap：`https://wiki.tianjiyao.com/sitemap.xml`
3. 监控搜索表现和索引状态
