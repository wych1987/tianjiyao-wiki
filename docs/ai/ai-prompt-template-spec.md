# AI提示词模板规范

AI 命理系统里，一个很常见的问题是所有任务都共用一套提示词。结果规则题被写得像长解释，高风险题被答得像普通问答，边界控制也经常被主任务风格冲掉。

这页提供一套可复用的提示词模板规范，目标不是给出某一个固定 prompt，而是定义不同任务该长什么骨架。

## 一、为什么要做模板拆分

因为不同任务的目标完全不同：

1. 规则题要清楚、稳定、少发挥
2. 解释题要结构化、可理解、贴场景
3. 高风险题要克制、边界清楚、必要时升级

如果三者共用同一模板，最容易发生风格污染。

## 二、建议至少拆成三套模板

1. 规则题模板
2. 解释题模板
3. 高风险题模板

## 三、规则题模板骨架

适用于排盘、识别、定位、结构确认类任务。

建议包含：

1. 任务定义
2. 允许使用的规则来源
3. 输出格式约束
4. 禁止无根据扩写

## 四、解释题模板骨架

适用于术语解释、结构说明、现实问题辅助理解。

建议包含：

1. 任务目标
2. 回答应先解释结构，再连接问题
3. 表达不确定性的方法
4. 禁止把辅助理解写成绝对结论

## 五、高风险题模板骨架

适用于健康、法律、投资、心理脆弱等场景。

建议包含：

1. 先识别高风险属性
2. 降低结论强度
3. 明确边界提示
4. 必要时请求补充信息或建议转向现实专业系统
5. 必要时触发人工复核条件

## 六、一个简化模板示例

### 规则题模板

1. 你当前执行的是规则确认任务
2. 仅基于给定规则和数据回答
3. 不补充未提供的信息
4. 输出保持简洁、结构化

### 解释题模板

1. 你当前执行的是结构解释任务
2. 先说明关键结构，再回应用户问题
3. 信息不足时必须说明不确定性
4. 不使用绝对化结论词

### 高风险题模板

1. 你当前处理的是高风险问题
2. 优先说明边界和不确定性
3. 不替代现实专业系统
4. 必要时建议补充信息、人工复核或转介

## 七、模板规范最常见的失败方式

1. 有分类，但最终仍然走同一套主提示词
2. 模板只有语气差别，没有任务动作差别
3. 高风险模板没有真正绑定升级条件

## 八、怎么判断模板是否真的分开了

1. 规则题是否显著更短、更克制
2. 解释题是否显著更重结构说明
3. 高风险题是否显著更重边界、补充信息和升级动作
4. 相同问题换模板时，输出风格和流程动作是否明显变化
