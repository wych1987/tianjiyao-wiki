# AI 命理与传统玄学

聚焦人工智能技术在传统命理学中的应用，覆盖机器学习、自然语言处理、知识图谱、大语言模型、知识库工作流、效果评估与伦理边界等方向，帮助读者理解 AI 命理到底能做什么、不能做什么，以及该如何更稳地使用它。

如果你此前把 AI 模块只理解为“技术炫点”，这一版更适合把它当成一个完整学习区：既看技术方法，也看评估框架、风险边界与真实应用条件。

如果你想先一页看清 AI 模块里哪些是概论页、哪些是方法页、哪些是落地页、哪些是评估与伦理页，可以直接进入 [AI模块完整索引：从机器学习到评估、伦理与协作流程](complete-index.md)。

## 🎯 学习路径

1. **基础认知** → 了解AI在命理学中的应用场景
2. **技术探索** → 学习具体的技术实现方法
3. **知识落地** → 理解古籍、规则与案例如何变成真正可检索、可约束的知识库
4. **效果评估** → 理解 AI 命理系统如何验证，不再只看“准不准”的口号
5. **伦理边界** → 理解哪些场景可以用、哪些场景必须回到现实专业系统
6. **协作流程** → 明确人工与模型在真实系统中的分工和复核机制
7. **未来展望** → 思考技术与文化的融合发展
