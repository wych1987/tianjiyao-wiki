# 财富事业专题导航

财富与事业是命理学习里最容易被问成“能不能发财”“适合做什么工作”的两个主题。但真正专业的学习，不应只追求结果判断，而要能说明：财富从哪里来、职业结构怎么跑、一个人的事业靠平台、资源、输出还是长期积累，以及某个阶段为什么会明显起势或卡住。

本页把站内与财富事业相关的内容，按跨体系专题重新整理出来。

## 一、三套体系在财富事业上的分工

1. 八字更适合看财星、官杀、食伤与承载力构成的长期结构
2. 紫微更适合看官禄、财帛、迁移、福德组成的职业与财富场域
3. 六爻更适合看当下求财、求职、升迁、项目、交易能否推进及何时应

### 八字重点问题

1. 财是怎么来的
2. 日主能不能任财任官
3. 事业靠官印、食伤还是资源结构
4. 哪步运势真正起势

4. 哪十年是真正的事业与财富主轴

### 六爻重点问题

1. 这个项目或财路现在能不能成
2. 这份工作能不能拿到
3. 什么时候出消息、签约、到岗、到账
4. 风险点是在竞争、流程还是承接能力
