# 命例库索引

随着站内内容逐渐增多，真正有价值的已经不只是“有没有案例”，而是“能不能快速按主题找到合适的案例去练”。本页的作用，就是把天机爻 Wiki 里已经形成的案例、命例和实战专题整理成一张统一索引，方便按体系、按主题、按学习阶段检索。

你可以把它理解为全站案例库的总目录。

## 一、如何使用这张索引

建议用下面三种方式使用本页：

1. 按体系找案例：适合正在专攻某一门的人
2. 按主题找案例：适合专门练婚姻、事业、财富、健康等专题的人
3. 按学习阶段找案例：适合想从基础命例过渡到复杂案例的人

## 七、建议的配套用法

如果你希望真正把案例学进去，建议一律按下面顺序练：

1. 先读对应体系的方法页
2. 再读相关专题页
3. 最后进入命例或实战页
4. 读完后自己总结出“这个案例为什么成立”

这样比单纯刷文章更容易形成判断力。
