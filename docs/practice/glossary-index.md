# 命理术语总表

随着站内内容越来越多，很多读者遇到的第一个障碍已经不是“找不到文章”，而是“同一个词在不同体系里到底是什么意思”。

这页的作用，就是把八字、紫微斗数、周易六爻里最常见、最容易混淆的术语集中整理成一张统一索引。你可以把它当成全站的查词中心。

## 一、这页怎么用

适合三种用法：

1. 看文章时遇到陌生词，先回到这里快速确认含义
2. 学完一个体系后，用这里做统一复盘，避免概念混淆
3. 做跨体系比较时，先确认同名词和近义词是否真是同一回事

## 二、八字常用术语

### 日主

八字分析的中心点，指日柱天干，代表命局中的“我”。判断旺衰、十神、取用，都要先回到日主。

延伸阅读：

- [八字命理入门教程](../bazi/introduction.md)
- [日主旺衰判断](../bazi/day-master-strength.md)

### 十神

十神是其他干支相对于日主形成的十种关系结构，包括比肩、劫财、食神、伤官、偏财、正财、七杀、正官、偏印、正印。它不是“十种事情”，而是十种作用方式。

延伸阅读：

- [十神体系详解](../bazi/ten-gods.md)

### 旺衰

旺衰指日主和全局气势的强弱状态，不是只看五行数量，而是综合月令、通根、透干、帮扶、耗泄、寒暖燥湿后的整体判断。

延伸阅读：

- [日主旺衰判断](../bazi/day-master-strength.md)

### 用神

用神是命局最需要的力量，用来调平结构、解决核心失衡。它不是单纯“喜欢什么五行”，而是根据命局问题来定。

延伸阅读：

- [用神喜忌与调候通关](../bazi/yongshen-tiaohou.md)

### 格局

格局是命局内部能量组织后形成的结构类型。判断格局时，不能只看某个神煞或单一标签，而要看整局气势是否成形、是否稳定、是否破格。

延伸阅读：

- [五行格局判断](../bazi/pattern-analysis.md)
- [特殊格命例专题](../bazi/special-pattern-case-studies.md)

### 调候

调候指从寒暖燥湿角度调整命局，使整体气候适合生发和承载。调候与扶抑不同，但在很多命局里要一起看。

延伸阅读：

- [用神喜忌与调候通关](../bazi/yongshen-tiaohou.md)

## 三、紫微常用术语

### 命宫

命宫是紫微斗数的核心宫位之一，用来观察人格主轴、人生起点和整体承接方式。它不是全部人生，但常是读盘入口。

延伸阅读：

- [命宫专题](../ziwei/life-palace-analysis.md)
- [排盘与安星规则](../ziwei/charting-rules.md)

### 星曜

星曜是紫微命盘中的符号系统，包括主星、辅曜、煞曜、杂曜等。它们不对应真实天体的物理影响，而是命理分析中的象征单元。

延伸阅读：

- [紫微斗数星曜解析](../ziwei/stars.md)
- [辅曜煞曜体系](../ziwei/assistant-stars.md)

### 宫位

宫位是人生不同领域的分区，例如命宫、夫妻宫、官禄宫、财帛宫、福德宫等。紫微读盘不是只看单宫，而是看宫位之间如何联动。

延伸阅读：

- [十二宫位详解](../ziwei/palaces.md)
- [三方四正与对宫体系](../ziwei/sanfang-sizheng.md)

### 三方四正

三方四正是紫微结构分析的核心框架，用来观察一个宫位与其对宫、三合宫之间的联动关系。专业判断通常不能只看单宫独断。

延伸阅读：

- [三方四正与对宫体系](../ziwei/sanfang-sizheng.md)

### 四化

四化指化禄、化权、化科、化忌，是紫微里极重要的动态信息。它不只是“吉凶标签”，而是能量流向、关注焦点与结构变化的提示。

延伸阅读：

- [四化飞星理论](../ziwei/four-transformations.md)

### 大限流年

大限和流年是紫微斗数中的阶段与年份观察层。大限看较长阶段的主轴变化，流年看具体年份如何触发结构中的重点。

延伸阅读：

- [大限流年总论](../ziwei/dynamic-fortune.md)
- [大限转折命例专题](../ziwei/dynamic-turning-point-case-studies.md)

## 四、六爻常用术语

### 用神

六爻中的用神，是针对当前问题所选取的核心判断对象。问婚姻、问求财、问工作、问疾病，用神的取法可能完全不同。

延伸阅读：

- [六爻断卦核心法则](../yijing/duan-gua-rules.md)
- [六爻分类占断总论](../yijing/classified-divination.md)

### 世应

世爻代表自己或发问方，应爻代表对方、对象或外部接口。很多关系题、合作题、求职题，都要先看世应关系是否成立。

延伸阅读：

- [六爻纳甲装卦体系](../yijing/najia-system.md)

### 动爻

动爻代表变化、推进、引发与事件进入运行状态。它常是判断转折和应期的重要入口。

延伸阅读：

- [六爻应期判断](../yijing/response-timing.md)
- [六爻应期案例专题](../yijing/response-timing-cases.md)

### 旬空

旬空表示当前未实、未定、未落地，常见于消息未到、事情未成、对象未表态等情况。判断应期时经常要看出空与填实。

延伸阅读：

- [六爻应期判断](../yijing/response-timing.md)

### 月破

月破表示当前阶段力量受损、事情不稳，往往提示当下难成或需要等待后续月份、日辰重新承接。

延伸阅读：

- [六爻断卦核心法则](../yijing/duan-gua-rules.md)

### 应期

应期是六爻里对“什么时候应”的判断。它不是单纯猜日期，而是判断事情的成熟节奏、触发层级和结果落点。

延伸阅读：

- [六爻应期判断](../yijing/response-timing.md)
- [六爻应期案例专题](../yijing/response-timing-cases.md)

## 五、最容易混淆的几个词

### 三套体系里的“用神”是不是同一个意思

不是。

1. 八字里的用神，是命局最需要的调节力量
2. 六爻里的用神，是当前问题的核心判断对象
3. 紫微斗数通常不以“用神”作为主框架

### 八字的“格局”和紫微的“格局”是不是同一个意思

也不是。

1. 八字格局偏命局气势组织方式
2. 紫微格局偏星曜组合形成的结构模式

虽然都叫格局，但判断逻辑不同。

### 六爻应期和八字流年是不是一回事

不是。

1. 八字流年偏长期时间结构中的年度作用
2. 紫微流年偏阶段场域中的年份触发
3. 六爻应期偏具体问题的近期或阶段性应验时间
