# 专题导航中心

当站内内容逐渐完整之后，读者最容易遇到的新问题不再是“有没有文章”，而是“我现在最关心的是婚姻、事业还是健康，该从哪里开始点进去”。

如果只按八字、紫微、六爻三个体系分别导航，适合系统学习；但如果读者带着现实问题进入站点，更需要一张按主题组织的总入口页。

这页就是为这个目的建立的。你可以把它理解为全站“按问题进入”的总目录。

## 一、这页怎么用

你可以按两种方式使用本页：

1. 如果你已经确定当前最关心的问题，就直接进入对应专题线
2. 如果你想建立整体框架，就先看本页，再回到[统一学习地图](learning-map.md)

简单说：

1. 学体系，用学习地图
2. 学问题，用专题导航中心

## 二、当前三条核心专题线

### [婚姻关系专题导航](marriage-topic-navigation.md)

适合关注以下问题的读者：

1. 婚姻结构稳不稳定
2. 配偶特征和关系模式怎么看
3. 合婚、合盘、婚后协同该如何系统学习
4. 眼前这段关系是否推进、复合或定局

专题内会串联：

1. 八字婚姻结构与合婚
2. 紫微夫妻宫、福德宫、双人合盘与婚后协同
3. 六爻感情占断与应期判断

### [财富事业专题导航](wealth-career-topic-navigation.md)

适合关注以下问题的读者：

1. 财路从哪里来
2. 职业路径与事业模式怎么判断
3. 哪个阶段会起势、卡住或转轨
4. 当前求职、项目、合作、求财能否推进

专题内会串联：

1. 八字的财星、官杀、食伤与长期事业结构
2. 紫微的官禄、财帛、迁移与大限转折
3. 六爻的求财、求职、项目推进与应期判断

### [健康风险专题导航](health-risk-topic-navigation.md)

适合关注以下问题的读者：

1. 体质和长期透支结构怎么看
2. 压力、恢复力和精神耗损如何判断
3. 哪些阶段更容易把风险放大
4. 当前病势、检查与恢复节奏如何观察

专题内会串联：

1. 八字的寒暖燥湿、五行偏枯与运势触发
2. 紫微的命宫、福德、疾厄与动态阶段
3. 六爻的疾病占断与时间转折

## 三、如果你不知道先看哪一条

可以用下面这组判断快速定位。

### 1. 你关心的是长期结构

优先读：

1. [婚姻关系专题导航](marriage-topic-navigation.md)
2. [财富事业专题导航](wealth-career-topic-navigation.md)
3. [健康风险专题导航](health-risk-topic-navigation.md)

再进入其中对应的八字和紫微部分。

### 2. 你关心的是眼前事件

优先读对应专题页中的六爻部分，尤其是：

1. 感情推进与复合
2. 求职、项目、求财进展
3. 病势、检查、恢复时间点

### 3. 你想做系统学习

建议顺序：

1. [统一学习地图](learning-map.md)
2. [专题导航中心](topic-navigation-center.md)
3. [命例库索引](case-library.md)

这样可以先建立主干，再进入具体问题，再进入案例训练。

## 四、专题导航和命例库的关系

这三类导航页不是重复，而是各有分工：

1. [统一学习地图](learning-map.md) 负责按阶段和体系组织学习顺序
2. [专题导航中心](topic-navigation-center.md) 负责按现实问题组织入口
3. [命例库索引](case-library.md) 负责把已经形成的案例和实战页集中起来

如果用一句话概括：

1. 学习地图解决“先后顺序”
2. 专题导航解决“问题入口”
3. 命例库解决“训练素材”

## 五、建议的使用方法

### 1. 初学者

先看[统一学习地图](learning-map.md)，再从本页挑一条最关心的专题线进入。

### 2. 已有基础的人

直接从本页进入感兴趣的专题，再顺着专题页里的链接深入到方法页和命例页。

### 3. 想做跨体系比较的人

最适合直接用本页，因为这里已经把同一问题下的八字、紫微、六爻入口放在一起。
