# 婚姻关系常见问题

关系主题最容易出现的，不是内容不够，而是问题太具体：我该先看合婚还是先看复合？为什么明明有感情却总推进不动？婚后协同到底算结构问题还是现实问题？这页把关系线里最常见的一批问题集中处理，适合作为关系专题的问答入口。

## 一、先分清你问的是哪一层

### Q1：我该先看合婚合盘，还是先看当前关系能不能推进？

先看你问的是长期结构，还是眼前事件。

1. 如果你想看双方长期匹配度、婚后协同和关系底盘，先看 [综合关系匹配：八字、紫微与六爻的多层判断框架](comprehensive-compatibility-analysis.md)
2. 如果你想看这段关系现在能不能推进、复合、见面或定下来，先看 [六爻婚姻感情占断](../yijing/marriage-divination.md)

### Q2：合婚是不是就是看两个人“合不合”？

不是。真正专业的关系判断，至少要分成三层：

1. 双方结构能不能长期承接
2. 关系模式与相处系统能不能运转
3. 当前阶段有没有推进接口

推荐阅读：

1. [八字合婚方法](../bazi/compatibility-analysis.md)
2. [紫微双人合盘方法](../ziwei/compatibility-analysis.md)
3. [综合关系匹配：八字、紫微与六爻的多层判断框架](comprehensive-compatibility-analysis.md)

## 二、关系为什么会卡住

### Q3：为什么两个人明明有感情，却总是推进不动？

很多关系不是没有感情，而是没有承接。

常见原因包括：

1. 双方结构并不支持长期关系落地
2. 关系模式存在明显错位
3. 当前阶段的现实条件和时间节奏不配合

推荐阅读：

1. [婚姻关系专题导航：八字、紫微、六爻的跨体系学习路线](marriage-topic-navigation.md)
2. [紫微斗数婚后协同专题：关系匹配度、分工模式与长期共处的专业读法](../ziwei/marital-collaboration-analysis.md)
3. [六爻婚姻感情占断](../yijing/marriage-divination.md)

### Q4：为什么八字看着能成，现实里却一直拖？

这通常不是冲突，而是层级不同。

1. 八字常在说长期婚姻结构和大方向
2. 紫微常在说关系模式和协同方式
3. 六爻常在说当前这一次推进有没有接口

推荐阅读：

1. [跨体系综合判断常见问题：八字、紫微与六爻如何一起使用](cross-system-faq.md)
2. [综合关系匹配：八字、紫微与六爻的多层判断框架](comprehensive-compatibility-analysis.md)

## 三、复合、离婚与关系边界

### Q5：复合问题应该看什么？

复合最关键的，不是还有没有感情，而是旧关系还有没有重新对接的条件。

重点应看：

1. 当前关系接口是否仍然存在
2. 双方节奏是否同步
3. 旧问题会不会被再次放大

推荐阅读：

1. [六爻婚姻感情占断](../yijing/marriage-divination.md)
2. [六爻应期判断](../yijing/response-timing.md)

### Q6：问离婚、分手边界这类题目，应该先看哪一套？

要分开看。

1. 想看这段关系从长期结构上是不是已经失去承接，先看合婚合盘和综合关系框架
2. 想看当前这次摊牌、分开、和解有没有动作接口，先看六爻婚姻卦

推荐阅读：

1. [综合关系匹配：八字、紫微与六爻的多层判断框架](comprehensive-compatibility-analysis.md)
2. [六爻婚姻感情占断](../yijing/marriage-divination.md)

## 四、婚后协同与现实问题

### Q7：婚后协同到底在看什么？

婚后协同看的是生活系统能不能运转，而不只是感情还在不在。

重点通常包括：

1. 分工方式
2. 财务节奏
3. 情绪承接
4. 空间与自由度需求
5. 现实压力来时能不能共担

推荐阅读：

1. [紫微斗数婚后协同专题：关系匹配度、分工模式与长期共处的专业读法](../ziwei/marital-collaboration-analysis.md)
2. [紫微双人合盘方法](../ziwei/compatibility-analysis.md)

### Q8：家人、异地、工作节奏这些现实因素，应该算关系问题还是外部问题？

它们都是关系问题的一部分，因为关系从来不是在真空里运行。

推荐阅读：

1. [夫妻宫与官禄宫专题](../ziwei/marriage-career-palaces.md)
2. [八字合婚方法](../bazi/compatibility-analysis.md)
