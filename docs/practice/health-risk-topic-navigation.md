# 健康风险专题导航

健康风险主题最容易被误用。真正专业的命理学习，不应该把它当成医疗替代，也不该把任何结构简单粗暴地对应为疾病结论。更稳的做法，是把健康主题理解为体质偏向、长期承压方式、恢复能力、阶段性风险窗口和生活节奏管理。

本页的目的，是把站内与健康风险相关的内容重新按专题串起来。

## 一、三套体系在健康风险上的分工

1. 八字更适合看五行偏枯、寒暖燥湿与长期透支结构
2. 紫微更适合看命宫、福德、疾厄构成的内在承受与恢复模式
3. 六爻更适合看当前病势、检查结果、治疗进程与风险应期

1. 五行哪里偏枯
2. 寒暖燥湿哪一端最失衡
3. 长期压力来自什么结构
