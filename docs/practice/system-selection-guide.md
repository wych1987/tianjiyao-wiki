# 体系选择指南

很多读者进入站点后第一反应不是“内容少”，而是“我到底该先看八字、紫微还是六爻”。如果入口选错，就很容易出现两个问题：要么用六爻去硬回答长期结构，要么拿八字去硬解释一件眼前小事，最后越看越乱。

这页的目标，就是先把三套体系的分工讲清楚，再把常见问题直接映射到合适入口。

## 一、先记住三句话

1. 八字更适合看长期结构、承载力和运势主轴
2. 紫微更适合看人格系统、人生场域和关系协同
3. 六爻更适合看一件具体事情当前能不能成、什么时候应

如果你只先记住这三句，很多入口混乱已经能解决一半。

## 二、按问题类型选体系

### 1. 想看自己是什么样的人，适合什么长期方向

1. [八字命理学习中心：四柱、十神与命例分析](../bazi/index.md)
2. [紫微斗数学习中心：星曜、宫位与命例分析](../ziwei/index.md)

这类题目更接近结构和人格，不是单次事件。

### 2. 想看关系匹配、婚后协同和双人结构

优先看：

### 4. 想看职业路径、财富结构和副业承接力

## 三、什么时候需要三套一起看

以下几类问题，通常最适合综合：

1. 婚姻关系
2. 财富事业
3. 健康恢复

原因很简单：

1. 八字负责长期结构
2. 紫微负责人格场与阶段变化
3. 六爻负责当前接口和时间节点

## 四、最常见的误区

1. 用六爻去回答“我一生适合什么”这类长期题目
2. 用八字去硬断“这周会不会收到消息”这类短期题目
3. 明明问的是双人关系，却只看单盘结构
4. 明明问的是当前推进，却一直停留在理论页不进入事件页

## 五、建议的使用顺序

1. 先用这页把问题分流
2. 再进入对应专题线或模块首页
3. 读完总纲页后，再决定要不要回主站做真实验证
