# 命理学习地图：八字、紫微、六爻进阶路线

现在这套 Wiki 的内容已经不算少，真正的问题不再是“有没有文章”，而是“先学什么、后学什么、什么时候切换体系”。如果没有一张统一学习地图，初学者很容易在八字、紫微、六爻之间反复跳转，读了很多页面，却没形成稳定框架。

本页的目标，就是把整个站点整理成一条可执行的进阶路线。你可以把它理解为全站导航版的课程表。

## 一、学习地图适合哪些人

这张地图主要适合三类读者：

1. 刚进入命理学习，不知道先从哪里开始的人
2. 已经学过一些零散内容，但体系混乱、路径不清的人
3. 想把八字、紫微、六爻三套系统串联起来的人

## 二、总学习原则

在进入具体路线前，先记住三个原则：

1. 先学底层理论，再学判断方法
2. 先把一套体系走通，再横向比较其他体系
3. 先会看结构，再追求命例和实战细节

如果顺序反过来，很容易出现“案例看得很热闹，自己下手不会断”的问题。

## 三、全站统一学习路线

建议按下面五个阶段推进。

### 第一阶段：打底层框架

这一阶段先不急着进入具体占断，而是先建立共同语言。

建议阅读顺序：

1. [阴阳理论基础](../theory/yinyang.md)
2. [天干详解](../theory/heavenly-stems.md)
3. [地支详解](../theory/earthly-branches.md)
4. [六十甲子体系](../theory/sixty-jiazi.md)
5. [自学路径指南](self-learning.md)

这一阶段完成后，至少应做到：

1. 理解阴阳五行、天干地支和干支组合的基本逻辑
2. 能分清“理论底层”和“具体术数系统”的区别
3. 建立长期学习而非零碎浏览的心态

### 第二阶段：先主修一门

三大主模块不建议一开始同时推进。更稳的做法是先选一门主修，把“理论页 → 方法页 → 专题页 → 命例页”完整走一遍。

你可以按自己的兴趣选择：

1. 想学人生结构、运势与关系判断，先从八字开始
2. 想学人格、宫位、关系与阶段变化，先从紫微开始
3. 想学具体问题、即时判断、应期与事件分析，先从六爻开始

### 第三阶段：补第二体系，形成比较视角

当你已经能完整读一套系统的基础盘后，再进入第二套体系。这一步的目标不是重复学习，而是建立比较能力。

例如：

1. 先学八字，再学紫微：能更好区分“结构”和“人格场”
2. 先学紫微，再学八字：能更好区分“宫位关系”和“干支气势”
3. 先学八字或紫微，再学六爻：更容易理解静态结构与动态事件的差别

这一步很适合配合阅读：

1. [八字与紫微对比：结构、人格与运势视角的差异](bazi-ziwei-comparison.md)
2. [命理术语总表：八字、紫微、六爻常用概念统一索引](glossary-index.md)

### 第四阶段：进入命例与应用层

到这一步，重点不再是记概念，而是训练拆盘和判断流程。

建议重点看：

1. 各模块的方法页
2. 各模块的高频专题页
3. 各模块命例页与应用页

### 第五阶段：做交叉学习与专题深化

这一阶段才适合做合盘、合婚、婚后协同、健康风险、考试应期、职业转折等复合专题。

也就是说，应用层永远排在结构层之后。

## 七、三体系如何衔接

当你三套体系都开始接触后，可以按下面方式建立互补视角：

1. 八字看人生底盘与长期结构
2. 紫微看人格主轴、关系场与阶段变化
3. 六爻看具体问题、当下决策与事件应期

如果用一句话概括：

1. 八字偏“命局结构”
2. 紫微偏“人生场域”
3. 六爻偏“具体事件”

这也是为什么进阶学习时，三套体系并不冲突，反而能互相补强。

## 八、不同目标的推荐路线

### 1. 目标是学会基础命理判断

建议路线：理论基础 → 八字基础线 → 八字方法线

### 2. 目标是学关系与婚恋判断

建议路线：八字婚姻线 + 紫微夫妻/福德线 + 紫微合盘线

### 3. 目标是学职业与财富判断

建议路线：八字财运事业线 + 紫微官禄财帛线 + 紫微财富命例线

### 4. 目标是学具体问题占断

建议路线：六爻基础线 → 六爻分类总论 → 具体专题 → 应期判断

### 5. 目标是系统研究三大体系

建议路线：

1. 先完成理论基础
2. 主修八字或紫微其中一套
3. 再补另一套结构类体系
4. 最后进入六爻事件判断体系

## 九、常见误区

### 1. 一开始三套一起学

容易信息过载，最后每套都只懂一点皮毛。

### 2. 只看案例，不学方法

会形成阅读快感，但不会真正下手分析。

### 3. 只学体系，不做比较

这样容易把每套系统都学得很窄，无法建立整体视角。

### 4. 长期停留在基础页

基础很重要，但不进入专题页和命例页，就难以形成真正判断力。

## 十、建议的使用方式

最稳妥的方式不是一次看完，而是把本页当成导航面板使用：

1. 先选一个主修方向
2. 按一条线连续读下去
3. 每完成一段，再回到本页切换下一阶段

这样比随手点文章高效得多。
