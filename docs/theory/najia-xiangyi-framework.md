# 纳甲类象框架

六爻里最难的一步，往往不是装卦，而是装完之后怎么把结构翻译成现实问题。很多人已经会安六亲、定六神、分世应，但一旦开始解释，就容易掉进两个极端：要么机械背表格，要么随意联想。

纳甲类象框架，就是为了解决这个问题。它讨论的不是“怎么装”，而是“装好之后怎么落地”。

## 一、纳甲类象到底在做什么

本质上，它是在回答三个问题：

1. 这个爻在现实里代表谁或什么
2. 它和问事核心是什么关系
3. 它接下来会怎样变化

这三个问题如果能答顺，六爻判断就会稳定很多。

## 二、先分清三个层次

### 1. 六亲是功能角色

六亲首先告诉你，这个爻在事件中扮演什么功能角色，例如资源、压力、支持、竞争、成果。

### 2. 六神是风格和状态

六神更多补充事件的表现方式，比如拖延、争论、隐情、风险、贵助。

### 3. 世应是立场关系

世应告诉你，这件事是从谁的视角展开，外部环境与自己之间形成什么互动。

## 三、为什么不能只背六亲表

例如妻财爻：

1. 问钱时，是财物和收入
2. 问生意时，是项目、客户、资源
3. 问感情时，可能是对象或关系投入

这说明六亲的象意不是固定名词，而是功能角色在不同问题中的具体落点。

## 四、六神怎么帮助你把事件看细

六神更像事件的气质修饰。

1. 朱雀常和信息、沟通、文书、争论有关
2. 白虎常和压力、损伤、手术、硬冲突有关
3. 玄武常和隐情、拖欠、暗中操作有关
4. 勾陈常和拖延、旧事、地产、黏滞有关

如果六亲告诉你“是什么”，六神往往补的是“以什么方式发生”。

## 五、世应为什么是事件场的骨架

很多人断卦时，最容易忽略世应，结果把结构看散。

更稳的做法是：

1. 先定世应，知道自己和外界的关系位置
2. 再看用神和关键爻
3. 最后再用六亲六神补细节

这样判断才不会只剩碎片化类象。

## 六、伏神飞神怎么进入类象

伏神很适合解释“事情存在，但还没有浮上来”的状态。

常见理解方式：

1. 用神伏，事情未明、条件未现、对象未出现
2. 伏神受制，表示信息被压住、被延后或被环境卡住
3. 飞神生扶或压制，会改变事情显化方式

这类判断，本质上也是类象，只是它更强调隐藏层。

## 七、纳甲类象和时间变化怎么结合

类象不是静态的，它会随着动爻、月日、冲合和出空入墓而变化。

所以真正专业的类象，不只是看“它是什么”，还要看“它什么时候出现、怎样变化、在什么条件下显化”。

## 八、最常见的误区

### 1. 机械背表

只会把六亲和现实对象一一硬对，遇到复杂问题就失效。

### 2. 只看六神，不看主体结构

六神只能修饰，不能替代用神、世应和旺衰。

### 3. 不区分问事场景

同一个爻，在求财、求职、问病、问感情里落点完全可能不同。
