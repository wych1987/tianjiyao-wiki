# 命理理论基础

系统整理阴阳五行、天干地支、六十甲子、五行对应、象意思维、气候调候、十神象意、宫位场域与纳甲类象框架，帮助读者为八字、紫微斗数和周易六爻建立统一底层逻辑。

如果你已经感觉模块页很多、知识点很散，那么理论区最适合做的事不是死背名词，而是先把底层关系走顺。学会理论层，后面再看八字的旺衰与调候、紫微的宫位场域、六爻的类象和纳甲，都会更稳。

如果你想先一页看完整个理论模块的结构、顺序和训练路径，可以直接进入 [理论基础完整索引：从阴阳五行到十神、宫位与纳甲方法](complete-index.md)。

## 📚 核心内容

### [阴阳五行](yinyang.md)
理解中国哲学的核心思想，掌握阴阳平衡与五行生克的基本原理。

### [天干详解](heavenly-stems.md)
系统学习十天干（甲乙丙丁戊己庚辛壬癸）的象征意义与应用规则。

### [地支详解](earthly-branches.md)
深入了解十二地支（子丑寅卯辰巳午未申酉戌亥）的深层含义与相互关系。

### [六十甲子](sixty-jiazi.md)
掌握干支组合的完整系统，理解六十甲子在命理学中的应用。

### [五行对应体系](wuxing-correspondences.md)
把五行与方位、四时、颜色、气候、人体和情志之间的对应关系整理成统一框架。

### [干支时空框架](ganzhi-time-space-framework.md)
理解天干地支如何同时表达时间、空间、层次与结构，是从记忆符号走向真正会用的关键一步。

### [象意思维基础](xiangyi-thinking.md)
解释类象和取象为什么是命理落地的关键，帮助把抽象符号翻译成现实问题。

### [气候与调候框架](climate-tiaohou-framework.md)
系统理解寒暖燥湿与调候逻辑，解决为什么有些结构要先看气候、再看强弱的问题。

### [十神象意框架](shishen-xiangyi-framework.md)
把比劫、食伤、财星、官杀、印星从标签改回关系语言，帮助八字分析真正落到现实问题。

### [十神象意误判案例](shishen-xiangyi-misjudgment-cases.md)
用反面案例训练十神为什么不能直接当标签，帮助建立更稳的八字落点意识。

### [宫位场域框架](palace-field-framework.md)
解释为什么位置不是孤点，而是主题、关系和资源共同形成的场域，适合接到紫微宫位与八字层次理解。

### [宫位场域误判案例](palace-field-misjudgment-cases.md)
用反面案例训练为什么只看单宫会出错，帮助把紫微宫位判断升级成场域判断。

### [纳甲类象框架](najia-xiangyi-framework.md)
把六亲、六神、世应与伏神如何落到具体事件结构讲清楚，适合接到六爻判断。

### [纳甲类象误判案例](najia-xiangyi-misjudgment-cases.md)
用反面案例训练六爻里“会装卦但不会落地”的典型错误，帮助提升事件解释稳定性。

4. [地支详解](earthly-branches.md)

6. [气候与调候框架](climate-tiaohou-framework.md)

11. [六十甲子](sixty-jiazi.md)

### 2. 如果你主要学八字

理论层里最重要的是：

1. 阴阳
2. 五行对应
3. 天干地支
4. 气候与调候
5. 十神象意
6. 干支时空框架

### 3. 如果你主要学紫微或六爻

也不要跳过理论层。紫微虽然不直接用八字式干支判断，六爻虽然有自己的卦爻系统，但它们依然要依托阴阳、五行、方位、场域和类象层次去理解。

## 🎯 学习价值

- 📖 **理论扎实**：系统掌握命理学的哲学基础
- 🧠 **逻辑严密**：理解古代先贤的思维方式
- 🔄 **触类旁通**：一通百通，应用于各个命理体系

## 💡 学习建议

1. **循序渐进**：从阴阳五行开始，逐步深入
2. **理解本质**：不要死记硬背，要理解背后的哲学思想
3. **联系实际**：将理论与八字、紫微、周易等实践相结合
4. **反复温习**：理论是基础，需要反复巩固

## 🔗 如何接到各模块

1. 学完阴阳、五行对应、气候调候和十神象意，再去看 [八字命理学习中心](../bazi/index.md) 会更容易理解旺衰、取用和现实落点
2. 学完宫位场域框架，再去看 [紫微斗数学习中心](../ziwei/index.md)，更容易理解宫位、三方四正与心理结构
3. 学完象意思维、纳甲类象和干支时空框架，再去看 [周易六爻学习中心](../yijing/index.md) 会更容易理解世应、六亲与应期类象
4. 学完理论层再回到 [命理实践指南](../practice/index.md)，更适合做跨体系学习

## 🔮 从理论走到实操

理论页不适合一上来就强导流，但当你已经知道自己更偏八字、紫微还是六爻时，可以先去对应工具做一次验证，再回来看理论，通常会更快建立手感。
