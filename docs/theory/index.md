# 理论基础

深入探究支撑传统命理学的核心哲学理论，理解中国古代智慧的底层逻辑。

## 📚 核心内容

### [阴阳五行](yinyang.md)
理解中国哲学的核心思想，掌握阴阳平衡与五行生克的基本原理。

### [天干详解](heavenly-stems.md)
系统学习十天干（甲乙丙丁戊己庚辛壬癸）的象征意义与应用规则。

### [地支详解](earthly-branches.md)
深入了解十二地支（子丑寅卯辰巳午未申酉戌亥）的深层含义与相互关系。

### [六十甲子](sixty-jiazi.md)
掌握干支组合的完整系统，理解六十甲子在命理学中的应用。

## 🎯 学习价值

- 📖 **理论扎实**：系统掌握命理学的哲学基础
- 🧠 **逻辑严密**：理解古代先贤的思维方式
- 🔄 **触类旁通**：一通百通，应用于各个命理体系

## 💡 学习建议

1. **循序渐进**：从阴阳五行开始，逐步深入
2. **理解本质**：不要死记硬背，要理解背后的哲学思想
3. **联系实际**：将理论与八字、紫微、周易等实践相结合
4. **反复温习**：理论是基础，需要反复巩固
