# 理论基础完整索引

如果你进入站点后发现理论页已经不少，但不知道该先学什么、哪些页是底层概念、哪些页是方法页、哪些页是反例训练页，那么最省时间的方式不是逐篇翻，而是先用这张完整索引把学习路线理顺。

这页把理论模块现有内容按“底层概念 → 中层方法 → 模块对接 → 误判训练 → 回接主模块”的方式重新整理，目标不是再解释一遍理论，而是让你知道每一页该在什么时候读、为什么要读。

## 一、先看哪一类内容

可以先按你的目标进入：

1. 想先补底层概念，看基础页
2. 想把抽象理论落到分析方法，看方法页
3. 想直接服务八字、紫微或六爻，看模块对接页
4. 想训练判断边界，看误判案例页
5. 想回到实战场景，看对应主模块首页
