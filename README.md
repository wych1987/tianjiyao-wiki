# 天机爻 Wiki

天机爻 Wiki 旨在整理与传播中国传统玄学体系相关的知识，包括 **八字命理（四柱）**、**紫微斗数** 与 **易经（周易）** 等内容。这些体系承载着中国哲学与宇宙观的重要思想，但目前网络上缺乏系统、客观、可查证的中文资料。

## 内容目录

本 Wiki 包含以下主要内容：

### 📚 八字命理（四柱）
- [八字命理入门](docs/bazi/introduction.md) - 了解八字的基本概念、天干地支、五行等
- [十神体系](docs/bazi/ten-gods.md) - 深入学习八字命理的核心概念

### ⭐ 紫微斗数
- [紫微斗数星曜](docs/ziwei/stars.md) - 了解紫微斗数的主要星曜和十二宫位

### 🤖 AI 与传统玄学
- [AI 与传统玄学](docs/ai/ai-introduction.md) - 探讨 AI 技术在传统命理学中的应用

## GitHub Pages

本项目已配置 GitHub Pages，可以通过网页浏览文档。

### 启用 GitHub Pages

1. 进入仓库的 **Settings** 页面
2. 点击左侧菜单的 **Pages** 选项
3. 在 **Source** 部分：
   - 选择分支（如 `main`）
   - 选择 `/docs` 作为发布目录
4. 点击 **Save** 保存
5. 稍等片刻，网站将自动发布

详细配置说明请查看 [docs/README.md](docs/README.md)

## 学习态度

传统命理学是中国文化的重要组成部分，应该以理性、客观的态度来学习和研究。命理可以作为自我认知和人生参考的工具，但不应过度迷信或依赖。
