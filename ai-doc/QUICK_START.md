# 🚀 快速使用指南

## 📋 可用命令

### 分析与检查
```bash
npm run analyze              # 分析所有文档质量
```

### 内容创建

#### 第一阶段：八字命理完善（进行中）
```bash
npm run create-missing       # 创建4个八字核心文档
```

#### 第二阶段：紫微斗数扩充
```bash
npm run create-ziwei         # 创建5个紫微文档
```

#### 第三阶段：AI玄学增强
```bash
npm run create-ai            # 创建4个AI技术文档
```

#### 第四阶段：实践应用
```bash
npm run create-practice      # 创建4个实践指导文档
```

#### 第五阶段：基础理论
```bash
npm run create-theory        # 创建4个理论基础文档
```

#### 一键创建全部
```bash
npm run create-all           # 执行所有阶段（需2-3小时）
```

### 优化工具
```bash
npm run optimize-links       # 优化内部链接
```
