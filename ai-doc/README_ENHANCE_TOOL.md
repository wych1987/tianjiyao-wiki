# 天机爻Wiki文档增强工具使用说明

## 📖 概述

这是一个基于DeepSeek API的专业Wiki文档生成和增强工具，旨在创建高质量、学术性强的传统文化知识内容。

## 🚀 快速开始

### 1. 环境准备

```bash
# 安装依赖
pip install requests pyyaml

# 设置API密钥（替换为您的真实密钥）
export DEEPSEEK_API_KEY="your_deepseek_api_key_here"
```

### 2. 基本使用

```bash
# 增强现有文档
python enhance_docs.py --api-key $DEEPSEEK_API_KEY --action enhance

# 创建新文档
python enhance_docs.py --api-key $DEEPSEEK_API_KEY --action create \
  --title "五行理论详解" --type "八字命理" --output "docs/bazi/wuxing.md"
```

## 📚 详细功能说明

### 增强现有文档模式

此模式会自动扫描并增强现有的markdown文档：

```bash
python enhance_docs.py \
  --api-key YOUR_API_KEY \
  --action enhance \
  --docs-path docs \
  --length 3000
```

**功能特点：**
- ✅ 自动检测文档类型（八字、紫微、AI玄学）
- ✅ 生成专业的学术内容
- ✅ 添加权威文献引用
- ✅ 优化SEO元素
- ✅ 创建备份文件
- ✅ 批量处理所有文档

### 创建新文档模式

创建全新的专业文档：

```bash
python enhance_docs.py \
  --api-key YOUR_API_KEY \
  --action create \
  --title "紫微斗数宫位详解" \
  --type "紫微斗数" \
  --output "docs/ziwei/palaces.md" \
  --length 4000
```

## 🔧 配置文件说明

### docs_config.py 配置项

- **QUALITY_STANDARDS**: 文档质量标准
- **TERMINOLOGY**: 专业术语词典
- **TRUSTED_SOURCES**: 可信来源列表
- **CONTENT_STRUCTURE**: 内容结构模板
- **SEO_CONFIG**: SEO优化配置
- **STYLE_GUIDE**: 语言风格指南

### 自定义配置

您可以根据需要修改配置文件：

```python
# 修改目标文档长度
QUALITY_STANDARDS["target_length"] = 4000

# 添加新的术语
TERMINOLOGY["八字命理"]["基础术语"].append("新术语")

# 添加可信来源
TRUSTED_SOURCES["学术机构"].append("新机构")
```

## ⚠️ 注意事项

### 使用建议

1. **API调用限制**: 注意DeepSeek API的调用频率限制
2. **内容审核**: 生成内容需要人工审核确认
3. **备份重要**: 自动创建备份文件，避免内容丢失
4. **渐进优化**: 建议分批次处理，逐步优化

### 最佳实践

1. **先测试**: 在小范围文档上测试效果
2. **分类处理**: 按文档类型分批处理
3. **质量优先**: 重视内容质量而非数量
4. **持续改进**: 根据效果调整提示词

## 🆘 故障排除

### 常见问题

**Q: API调用失败怎么办？**
A: 检查API密钥、网络连接和调用频率限制

**Q: 生成内容质量不理想？**
A: 调整提示词参数，或增加更具体的要求

**Q: 文档格式不正确？**
A: 检查Markdown语法和YAML front matter格式

**Q: 如何避免内容重复？**
A: 为每种文档类型设计不同的提示词模板
